## 1. A* 평가함수 계산

#### A* 탐색에서 평가함수는 다음과 같다.

#### 𝑓(𝑛)=𝑔(𝑛)+ℎ(𝑛)

어떤 현재 노드 n에 대해 다음 값이 주어졌다.

```
● 지금까지의 실제 비용 𝑔(𝑛)= 5
● 목표까지의 휴리스틱 추정값 ℎ(𝑛)= 3
```
1. 이 노드의 𝑓(𝑛) 값을 계산하시오.
2. 같은 시점에 열린 노드들 중 평가값이 각각 10, 7, 8 이라면, A는 위 노드를 먼저
    확장하는가? 이유를 함께 쓰시오.

#### 정답

#### 1. 𝑓(𝑛)=𝑔(𝑛)+ℎ(𝑛)= 5 + 3 = 8 이다.

2. A*는 평가값 f(n) 이 가장 작은 노드를 우선 확장한다. 이 노드의 값은 8 이므로, 열린
    노드들 중 값이 7 인 노드가 있다면 그 노드가 먼저 확장된다. 따라서 이 노드가 가장
    먼저 확장되지는 않는다.

## 5. Backpropagation in a 2-Layer MLP

다음과 같은 2-layer MLP 모델을 고려하시오.

### Model

z₁ = w₁ x + b₁

a₁ = ReLU(z₁)

z₂ = w₂ a₁ + b₂

ŷ = z₂


Loss function:

L = 1/2 (ŷ − y)²

### Given

x = 2, y = 1

w₁ = 1, b₁ = 0

w₂ = 2, b₂ = 0

learning rate η = 0.

### (5-1) Forward Pass

#### 다음을 계산하시오:

```
● z₁
```
```
● a₁
```
```
● z₂
```
```
● ŷ
```
#### ● L

### (5-2) Backpropagation

다음 gradient를 계산하시오:

```
● ∂L/∂w₂
```
```
● ∂L/∂b₂
```
```
● ∂L/∂w₁
```
```
● ∂L/∂b₁
```

(ReLU 미분은 z > 0 일 때 1, 그렇지 않으면 0 으로 가정한다.)

### (5-3) Parameter Update

Gradient descent를 사용하여 업데이트된 파라미터를 구하시오.

w ← w − η ∂L/∂w

#### 정답

### (5-1) Forward Pass

**z₁ = w₁ x + b₁ = 1 × 2 + 0 = 2**

**a₁ = ReLU(2) = 2**

**z₂ = w₂ a₁ + b₂ = 2 × 2 + 0 = 4**

**ŷ = 4**

**L = 1/2 (4 − 1)² = 1/2 × 9 = 4.**

### (5-2) Backpropagation

#### 먼저 출력층부터 계산한다.

**∂L/∂ŷ = (ŷ − y) = (4 − 1) = 3**

### Output layer

**∂L/∂w₂ = ∂L/∂ŷ × ∂ŷ/∂w₂**

**= 3 × a₁ = 3 × 2 = 6**

**∂L/∂b₂ = ∂L/∂ŷ × 1 = 3**

### Hidden layer


**∂L/∂a₁ = ∂L/∂ŷ × w₂ = 3 × 2 = 6**

**ReLUʼ(z₁) = 1 (z₁ = 2 > 0)**

**∂L/∂z₁ = 6 × 1 = 6**

**∂L/∂w₁ = ∂L/∂z₁ × x = 6 × 2 = 12**

**∂L/∂b₁ = ∂L/∂z₁ × 1 = 6**

### (5-3) Parameter Update

**w₂_new = 2 − 0.1 × 6 = 1.4**

**b₂_new = 0 − 0.1 × 3 = -0.3**

**w₁_new = 1 − 0.1 × 12 = -0.2**

**b₁_new = 0 − 0.1 × 6 = -0.6**

**(** 최종 답 **)**

**w₁ = -0.2**

**b₁ = -0.6**

**w₂ = 1.4**

**b₂ = -0.3**

## 6. 층별 선훈련 (Layer-wise Pretraining)

딥러닝 초기 발전에서 사용된 층별 선훈련(layer-wise pretraining) 방법에 대해 설명하고,
이 방법이 왜 혁신적이었는지 간략히 서술하시오.

#### 정답

층별 선훈련(layer-wise pretraining)은 신경망을 한 번에 학습하는 대신,

각 층을 순차적으로 학습한 후 전체 네트워크를 미세조정(fine-tuning)하는 방법이다.


주로 Restricted Boltzmann Machine(RBM)이나 autoencoder를 사용하여 비지도
학습으로 초기 가중치를 설정하였다.

이 방법은 깊은 신경망에서 발생하는 gradient vanishing 문제를 완화하고,

랜덤 초기화 대신 의미 있는 초기값을 제공함으로써 학습을 안정적으로 만들었다.

따라서 층별 선훈련은 딥러닝이 효과적으로 학습될 수 있도록 한 초기 핵심 기술로서
혁신적이었다.

채점 포인트

```
● layer-wise 학습 설명 ✔
```
```
● unsupervised pretraining 언급 ✔
```
```
● vanishing gradient 완화 ✔
```
```
● deep network 학습 가능하게 함 ✔
```
