### 문제 5. Backpropagation in a 2-Layer MLP

[정답]

### (5-1) Forward Pass

**z₁ = w₁ x + b₁ = 1 × 2 + 0 = 2**

**a₁ = ReLU(2) = 2**

**z₂ = w₂ a₁ + b₂ = 2 × 2 + 0 = 4**

**ŷ = 4**

**L = 1/2 (4 − 1)² = 1/2 × 9 = 4.**

### (5-2) Backpropagation

#### 먼저 출력층부터 계산한다.

**∂L/∂ŷ = (ŷ − y) = (4 − 1) = 3**

### Output layer

**∂L/∂w₂ = ∂L/∂ŷ × ∂ŷ/∂w₂**

**= 3 × a₁ = 3 × 2 = 6**

**∂L/∂b₂ = ∂L/∂ŷ × 1 = 3**

### Hidden layer


**∂L/∂a₁ = ∂L/∂ŷ × w₂ = 3 × 2 = 6**

**ReLUʼ(z₁) = 1 (z₁ = 2 > 0)**

**∂L/∂z₁ = 6 × 1 = 6**

**∂L/∂w₁ = ∂L/∂z₁ × x = 6 × 2 = 12**

**∂L/∂b₁ = ∂L/∂z₁ × 1 = 6**

### (5-3) Parameter Update

**w₂_new = 2 − 0.1 × 6 = 1.4**

**b₂_new = 0 − 0.1 × 3 = -0.3**

**w₁_new = 1 − 0.1 × 12 = -0.2**

**b₁_new = 0 − 0.1 × 6 = -0.6**

**(** 최종 답 **)**

**w₁ = -0.2**

**b₁ = -0.6**

**w₂ = 1.4**

**b₂ = -0.3**

### 문제 6. 층별 선훈련 (Layer-wise Pretraining)

[정답]

층별 선훈련(layer-wise pretraining)은 신경망을 한 번에 학습하는 대신,

각 층을 순차적으로 학습한 후 전체 네트워크를 미세조정(fine-tuning)하는 방법이다.


주로 Restricted Boltzmann Machine(RBM)이나 autoencoder를 사용하여 비지도
학습으로 초기 가중치를 설정하였다.

이 방법은 깊은 신경망에서 발생하는 gradient vanishing 문제를 완화하고,

랜덤 초기화 대신 의미 있는 초기값을 제공함으로써 학습을 안정적으로 만들었다.

따라서 층별 선훈련은 딥러닝이 효과적으로 학습될 수 있도록 한 초기 핵심 기술로서
혁신적이었다.


[채점 포인트]

```
● layer-wise 학습 설명 ✔
```
```
● unsupervised pretraining 언급 ✔
```
```
● vanishing gradient 완화 ✔
```
```
● deep network 학습 가능하게 함 ✔
```

### 문제 21. Attention 메커니즘

[정답]

**(1)**

기본 Encoder-Decoder 구조에서는 입력 시퀀스 전체 정보를 **고정된 길이의 하나의 벡터(context vector)** 로 압축하여 디코더에 전달함.
입력 문장이 길거나 복잡할 경우, 이 하나의 벡터만으로 모든 정보를 담기 어렵고, 결과적으로 정보 손실이 발생함 → **성능 저하**

**(2)**

Attention은 **입력 시퀀스의 모든 hidden state**를 참조하며, 각 디코딩 스텝마다 현재 디코더 상태와 입력 시퀀스의 각 hidden state 간의 유사도(attention score)를 계산함.
이 유사도를 기반으로 가중합(context vector)을 동적으로 계산하여, 매 스텝마다 중요한 입력 정보를 선택적으로 반영함 → **정보 손실 없이 더 정확한 출력 가능**

### 문제 25. Pairwise Preference와 RLHF / DPO

[정답]

**(1)** 만들 수 있는 pairwise preference label은 다음과 같다.

$$
y_2 > y_1, \quad y_1 > y_3, \quad y_2 > y_3
$$

**(2)** RLHF에서는 위 pairwise preference 데이터를 이용해 Reward Model을 학습한다.
Reward Model은 같은 입력 $x$에 대해 사람이 더 선호한 답변에는 더 높은 reward를, 덜 선호한 답변에는 더 낮은 reward를 주도록 학습된다.

그 후 언어모델은 이 Reward Model의 점수가 높은 답변을 생성하도록 **PPO와 같은 강화학습 방식**으로 최적화된다.

**(3)** DPO에서는 **별도의 Reward Model을 학습하지 않는다.**
대신 선호된 답변 $y^+$의 생성 확률은 높이고, 거부된 답변 $y^-$의 생성 확률은 낮추도록 **언어모델을 직접 학습**한다.

### 문제 26. RLHF의 PPO 최적화 — Reward Model & KL Penalty

[정답]

**(1)**

Reward Model은 **사람이 선호하는 답변일수록 높은 점수를 주도록** 학습되며, 언어모델이 더 좋은 답변을 생성하도록 **학습 방향을 제공**한다.

**(2)**

KL divergence penalty는 학습된 모델이 **기존 사전학습 모델의 분포에서 너무 크게 벗어나지 않도록 제한**한다. 이를 통해 **학습 안정성을 유지**하고 **비정상적인 출력이 생성되는 것을 방지**한다.
