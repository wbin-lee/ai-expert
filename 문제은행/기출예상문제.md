## 1л¶Җ. мқёкіөм§ҖлҠҘ кё°мҙҲ (260406~260409 / мһҘлі‘нғҒ көҗмҲҳлӢҳ)

### л¬ём ң 1. CNN м¶ңл Ҙ нҒ¬кё° л°Ҹ нҢҢлқјлҜён„° мҲҳ кі„мӮ°

лӢӨмқҢкіј к°ҷмқҖ CNN лӘЁлҚём—җ MNIST мқҙлҜём§ҖлҘј мһ…л Ҙн•ңлӢӨ.

```
в—Ҹ мһ…л Ҙ мқҙлҜём§Җ нҒ¬кё° : 1 Г— 28 Г— 28 (мұ„л„җ=1, 28Г—28 н”Ҫм…Җ)
в—Ҹ Conv2D : н•„н„° 32к°ң, м»Өл„җ 3Г—3, stride=1, padding=0
в—Ҹ MaxPool2D : м»Өл„җ 2Г—2, stride=2
в—Ҹ Conv2D : н•„н„° 64к°ң, м»Өл„җ 3Г—3, stride=1, padding=0
в—Ҹ MaxPool2D : м»Өл„җ 2Г—2, stride=2
в—Ҹ Flatten нӣ„ FC вҶ’ 10к°ң нҒҙлһҳмҠӨ м¶ңл Ҙ
```

лӢӨмқҢмқ„ мҲңм„ңлҢҖлЎң кі„мӮ°н•ҳмӢңмҳӨ. (м¶ңл Ҙ нҒ¬кё°лҠ” м •мҲҳл¶Җ `вҢҠ вҢӢ` лЎң лӮҙлҰјн•ңлӢӨ.)

1. мІ« лІҲм§ё Conv2D м§Ғнӣ„мқҳ м¶ңл Ҙ н…җм„ңмқҳ кіөк°„ нҒ¬кё°(HГ—W)
2. мІ« лІҲм§ё MaxPool2D м§Ғнӣ„мқҳ м¶ңл Ҙ н…җм„ңмқҳ кіөк°„ нҒ¬кё°(HГ—W)
3. л‘җ лІҲм§ё Conv2D м§Ғнӣ„мқҳ м¶ңл Ҙ н…җм„ңмқҳ кіөк°„ нҒ¬кё°(HГ—W)
4. л‘җ лІҲм§ё MaxPool2D м§Ғнӣ„мқҳ м¶ңл Ҙ н…җм„ңмқҳ кіөк°„ нҒ¬кё°(HГ—W)
5. Flatten м§Ғнӣ„мқҳ лІЎн„° м°Ёмӣҗ мҲҳ (мұ„л„җ Г— H Г— W)
6. м „мІҙ FC кі„мёөмқҳ нҢҢлқјлҜён„° мҲҳ(к°ҖмӨ‘м№ҳ + л°”мқҙм–ҙмҠӨ). лӢЁ, м§Ғм „ Flatten м°Ёмӣҗкіј 10к°ң м¶ңл Ҙм—җ лҢҖн•ң fully-connected кі„мӮ°мқҙлӢӨ.

### л¬ём ң 2. RNNмқҳ н•ңкі„мҷҖ LSTM кІҢмқҙнҠё кө¬мЎ°

к°•мқҳм—җм„ңлҠ” мӢңнҖҖмҠӨ лҚ°мқҙн„°лҘј мІҳлҰ¬н•ҳлҠ” лӘЁлҚёлЎң RNNкіј к·ё к°ңм„  лӘЁлҚёмқё LSTMмқ„ мҶҢк°ңн•ҳмҳҖлӢӨ.

1. кё°ліё RNNмқҙ кёҙ мӢңнҖҖмҠӨлҘј мІҳлҰ¬н•  л•Ң л°ңмғқн•ҳлҠ” лҢҖн‘ңм Ғмқё н•ҷмҠө л¬ём ңлҘј 1к°ң м“°кі , мҷң л°ңмғқн•ҳлҠ”м§Җ н•ң мӨ„лЎң м„ӨлӘ…н•ҳмӢңмҳӨ.
2. LSTMмқҙ лҸ„мһ…н•ң 3к°ңмқҳ кІҢмқҙнҠё(gate) мқҙлҰ„мқ„ лӘЁл‘җ м“°кі , к°Ғ кІҢмқҙнҠёк°Җ мҲҳн–үн•ҳлҠ” м—ӯн• мқ„ н•ң мӨ„м”© м“°мӢңмҳӨ.
3. LSTMмқҙ кё°ліё RNN лҢҖл№„ к°–лҠ” мқҙм җмқ„ "мһҘкё° мқҳмЎҙм„ұ(long-term dependency)" к°ңл…җкіј м—°кІ°н•ҳм—¬ 1~2мӨ„лЎң м„ңмҲ н•ҳмӢңмҳӨ.

### л¬ём ң 3. Scaled Dot-Product Attention кі„мӮ°

Transformerмқҳ н•өмӢ¬ м—°мӮ°мқё Scaled Dot-Product AttentionмқҖ лӢӨмқҢкіј к°ҷмқҙ м •мқҳлҗңлӢӨ.

```
Attention(Q, K, V) = softmax( QВ·KбөҖ / вҲҡd_k ) В· V
```

н•ң к°ңмқҳ мҝјлҰ¬(Query) лІЎн„°мҷҖ 2к°ңмқҳ нӮӨ/к°’(Key, Value) лІЎн„°к°Җ лӢӨмқҢкіј к°ҷмқҙ мЈјм–ҙмЎҢлӢӨ. (нӮӨмҷҖ к°’мқҳ м°Ёмӣҗ `d_k = 2`)

```
в—Ҹ Q  = [1, 0]
в—Ҹ KвӮҒ = [1, 0],   VвӮҒ = [1, 2]
в—Ҹ KвӮӮ = [0, 1],   VвӮӮ = [3, 4]
```

лӢӨмқҢмқ„ мҲңм„ңлҢҖлЎң кө¬н•ҳмӢңмҳӨ. (softmax кі„мӮ° мӢң `e вүҲ 2.718` мқ„ мӮ¬мҡ©н•ҳкі , мөңмў… к°’мқҖ мҶҢмҲҳ л‘ҳм§ё мһҗлҰ¬к№Ңм§Җ л°ҳмҳ¬лҰјн•ҳм—¬ м“°мӢңмҳӨ.)

1. QВ·KвӮҒ л°Ҹ QВ·KвӮӮ мқҳ лӮҙм Ғ м җмҲҳ
2. вҲҡd_k лЎң лӮҳлҲҲ нӣ„мқҳ мҠӨмјҖмқјлҗң м җмҲҳ
3. л‘җ м җмҲҳм—җ лҢҖн•ң softmax к°ҖмӨ‘м№ҳ ОұвӮҒ, ОұвӮӮ
4. мөңмў… Attention м¶ңл Ҙ лІЎн„° (= ОұвӮҒВ·VвӮҒ + ОұвӮӮВ·VвӮӮ)
5. мң„ мӢқм—җм„ң вҲҡd_k лЎң лӮҳлҲ„м–ҙ "мҠӨмјҖмқј"н•ҳлҠ” мқҙмң лҘј н•ң мӨ„лЎң м„ңмҲ н•ҳмӢңмҳӨ.

### л¬ём ң 4. BERT vs GPT кө¬мЎ°В·н•ҷмҠө л№„көҗмҷҖ In-Context Learning

лҢҖн‘ңм Ғмқё мӮ¬м „н•ҷмҠө м–ём–ҙлӘЁлҚёмқё **BERT**мҷҖ **GPT**лҠ” лӘЁл‘җ TransformerлҘј кё°л°ҳмңјлЎң н•ҳм§Җл§Ң кө¬мЎ°мҷҖ н•ҷмҠө л°©мӢқм—җ м°Ёмқҙк°Җ мһҲлӢӨ.

1. BERTмҷҖ GPTлҠ” к°Ғк°Ғ Transformerмқҳ м–ҙл–Ө л¶Җл¶„(Encoder / Decoder)мқ„ мӮ¬мҡ©н•ҳлҠ”м§Җ м“°мӢңмҳӨ.
2. л‘җ лӘЁлҚёмқҳ мӮ¬м „н•ҷмҠө лӘ©н‘ң(pre-training objective)лҘј к°Ғк°Ғ м“°мӢңмҳӨ.
3. л‘җ лӘЁлҚёмқҳ л¬ёл§Ҙ нҢҢм•… л°©н–Ҙ(Bidirectional / Unidirectional)мқ„ к°Ғк°Ғ м“°мӢңмҳӨ.
4. лӢӨмқҢ л‘җ нғңмҠӨнҒ¬лҠ” BERTмҷҖ GPT мӨ‘ м–ҙлҠҗ мӘҪм—җ лҚ” м Ғн•©н•ңм§Җ к°Ғк°Ғ мқҙмң мҷҖ н•Ёк»ҳ м“°мӢңмҳӨ.
   - (к°Җ) л¬ёмһҘ л№Ҳм№ё мұ„мҡ°кё° / л¬ёмһҘ л¶„лҘҳ / к°ңмІҙлӘ… мқёмӢқ
   - (лӮҳ) мЈјм–ҙм§„ н”„лЎ¬н”„нҠёлҘј мқҙм–ҙ л°ӣм•„ кёҙ кёҖмқ„ мғқм„ұн•ҳкё°
5. GPT-3 мқҙнӣ„ LLMм—җм„ң к°•мЎ°лҗң **In-Context Learning (Few-shot Learning)** мқҳ к°ңл…җмқ„ н•ңл‘җ мӨ„лЎң м„ңмҲ н•ҳмӢңмҳӨ.

### л¬ём ң 5. Scaled Dot-Product Attention кі„мӮ°

Transformerмқҳ attentionмқҖ лӢӨмқҢ мӢқмңјлЎң кі„мӮ°лҗңлӢӨ.

```
Attention(Q, K, V) = softmax(QK^T / sqrt(d_k))V
```

лӢӨмқҢ лІЎн„°к°Җ мЈјм–ҙм ё мһҲлӢӨ.

```
Q  = [1, 0]
K1 = [1, 0],   V1 = [2, 0]
K2 = [0, 1],   V2 = [0, 2]
d_k = 2
```

лӢӨмқҢмқ„ кі„мӮ°н•ҳмӢңмҳӨ. лӢЁ, `sqrt(2)=1.414`, `e^0.707вүҲ2.03`, `e^0=1`лЎң кі„мӮ°н•ҳкі , мөңмў… лӢөмқҖ мҶҢмҲҳ л‘ҳм§ё мһҗлҰ¬к№Ңм§Җ м“°мӢңмҳӨ.

1. `QВ·K1`, `QВ·K2` лӮҙм Ғ м җмҲҳ
2. `sqrt(d_k)`лЎң лӮҳлҲҲ scaled score
3. softmax attention weight `alpha1`, `alpha2`
4. мөңмў… Attention м¶ңл Ҙ лІЎн„°
5. `sqrt(d_k)`лЎң лӮҳлҲ„лҠ” мқҙмң лҘј н•ң мӨ„лЎң м„ӨлӘ…н•ҳмӢңмҳӨ.

### л¬ём ң 6. BERTмҷҖ GPT л№„көҗ

BERTмҷҖ GPTлҠ” лӘЁл‘җ Transformer кё°л°ҳ лӘЁлҚёмқҙм§Җл§Ң кө¬мЎ°мҷҖ н•ҷмҠө л°©мӢқмқҙ лӢӨлҘҙлӢӨ. лӢӨмқҢ л¬јмқҢм—җ лӢөн•ҳмӢңмҳӨ.

1. BERTмҷҖ GPTк°Җ к°Ғк°Ғ Transformerмқҳ м–ҙлҠҗ л¶Җл¶„мқ„ мӮ¬мҡ©н•ҳлҠ”м§Җ м“°мӢңмҳӨ.
2. BERTмҷҖ GPTмқҳ мӮ¬м „н•ҷмҠө лӘ©н‘ңлҘј к°Ғк°Ғ м“°мӢңмҳӨ.
3. BERTмҷҖ GPTмқҳ л¬ёл§Ҙ мІҳлҰ¬ л°©н–Ҙмқ„ к°Ғк°Ғ м“°мӢңмҳӨ.
4. лӢӨмқҢ нғңмҠӨнҒ¬м—җ лҚ” м Ғн•©н•ң лӘЁлҚёмқ„ BERT/GPT мӨ‘м—җм„ң кі лҘҙкі  мқҙмң лҘј м“°мӢңмҳӨ.
