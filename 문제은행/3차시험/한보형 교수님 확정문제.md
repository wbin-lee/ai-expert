## 문제 1

R-CNN보다 Fast R-CNN이 빠른 이유를 서술하시오.

### 답안

R-CNN은 region proposal마다 CNN을 개별 실행(~2,000회)하지만, Fast R-CNN은 이미지 전체를 CNN에 한 번만 통과시켜 feature map을 생성한 뒤 RoI Pooling으로 각 영역의 특징을 추출한다. CNN 연산 횟수를 대폭 줄여 속도가 크게 향상된다.

## 문제 2

RetinaNet에서 사용된 **Focal Loss**의 수식은 다음과 같다.

$$FL(p_t) = -(1 - p_t)^\gamma \cdot \log(p_t)$$

`γ = 0`일 때 Focal Loss와 동일해지는 기존 손실 함수의 명칭을 쓰고, `γ > 0`일 때 잘 분류된 예제(well-classified examples)에 미치는 효과를 서술하시오.

### 답안

- `γ = 0`이면 **Cross Entropy(CE) 손실**과 동일하다.
- `γ > 0`이면 잘 분류된 예제(`p_t`가 높은 샘플)의 손실이 크게 줄어(down-weighted), 학습이 어려운 예제에 집중된다.

## 문제 3

DETR의 매칭 비용(matching cost)은 클래스 확률 항과 bounding box 항(L1 loss + GIoU loss)으로 정의된다.

$$C(GT_i, Pred_j) = -p_j(c_i) + \lambda_{L1} \cdot L1(b_i, b_j) + \lambda_{iou} \cdot L_{giou}(b_i, b_j)$$

- $p_j(c_i)$: $Pred_j$가 $GT_i$의 클래스 $c_i$에 부여한 확률 (각 prediction의 클래스 확률 합은 1)
- L1 loss = $\|b_i - b_j\|_1$, 단 박스는 정규화된 `(cx, cy, w, h)` 형식으로 계산
- GIoU loss = $1 - GIoU$, $GIoU = IoU - \dfrac{A_C - A_\cup}{A_C}$ ($C$: 두 박스를 감싸는 최소 박스, $A_\cup$: 합집합 넓이) — 박스 기하로 계산
- 가중치: $\lambda_{L1} = 5$, $\lambda_{iou} = 2$ (DETR 기본값), 이미지 해상도 `10 × 10`

아래 표의 ground-truth 2개와 prediction 2개에 대해, 두 일대일 매칭 ㉠, ㉡ 중 전체 cost 합이 더 작은(최적인) 것을 계산을 통해 고르고, 그 최소 total cost를 구하시오.

- 매칭 ㉠: GT1–P1, GT2–P2
- 매칭 ㉡: GT1–P2, GT2–P1

|  | 클래스 | corners (x1, y1, x2, y2) | norm (cx, cy, w, h) | p(A) | p(B) |
|---|---|---|---|---:|---:|
| GT1 | A | (0, 0, 2, 2) | (0.1, 0.1, 0.2, 0.2) | - | - |
| GT2 | B | (4, 4, 8, 8) | (0.6, 0.6, 0.4, 0.4) | - | - |
| P1 | - | (0, 0, 2, 4) | (0.1, 0.2, 0.2, 0.4) | 0.8 | 0.2 |
| P2 | - | (4, 4, 8, 6) | (0.6, 0.5, 0.4, 0.2) | 0.3 | 0.7 |

### 답안

각 쌍의 항 계산 (L1 loss는 `(cx, cy, w, h)` 절대차의 합, GIoU loss = `1 − GIoU`)

- **C(GT1, P1)**: L1 = `0 + 0.1 + 0 + 0.2 = 0.3`, GIoU = `0.5` → GIoU loss `0.5`

$$= -0.8 + 5(0.3) + 2(0.5) = -0.8 + 1.5 + 1.0 = 1.7$$

- **C(GT2, P2)**: L1 = `0 + 0.1 + 0 + 0.2 = 0.3`, GIoU = `0.5` → GIoU loss `0.5`

$$= -0.7 + 1.5 + 1.0 = 1.8$$

- **C(GT1, P2)**: L1 = `0.5 + 0.4 + 0.2 + 0 = 1.1`, 겹침 없음 IoU = `0`, $A_\cup = 12$, $A_C = 48$ → GIoU = `−0.75` → GIoU loss `1.75`

$$= -0.3 + 5(1.1) + 2(1.75) = -0.3 + 5.5 + 3.5 = 8.7$$

- **C(GT2, P1)**: L1 = `0.5 + 0.4 + 0.2 + 0 = 1.1`, 겹침 없음 IoU = `0`, $A_\cup = 24$, $A_C = 64$ → GIoU = `−0.625` → GIoU loss `1.625`

$$= -0.2 + 5(1.1) + 2(1.625) = -0.2 + 5.5 + 3.25 = 8.55$$

매칭별 total cost:

- 매칭 ㉠ = `1.7 + 1.8 = 3.5`
- 매칭 ㉡ = `8.7 + 8.55 = 17.25`

→ 최적 매칭은 **㉠ (GT1–P1, GT2–P2)**, 최소 total cost = **3.5**
