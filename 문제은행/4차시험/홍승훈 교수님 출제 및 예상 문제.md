### (5) 확정문제별 심층 분석

#### Q1 — Reparameterization trick (균일분포 버전)

**문제 구조**: $a = \text{ReLU}(Wx)$ → $b \sim \text{Unif}(-a, a)$ → $y = W'b$

**시험이 검증하는 것**
- 샘플링 연산은 **stochastic**이므로 $b$에 대한 gradient가 **끊긴다**는 이해
- 해결: $b = a \cdot e$, $e \sim \text{Unif}(-1, 1)$ 로 **deterministic path** ($a$)와 **고정 분포 샘플** ($e$) 분리
- VAE의 $Z = \mu + \sigma \odot \epsilon$, $\epsilon \sim \mathcal{N}(0,I)$ 와 **동일한 원리** — 가우시안이 아니어도 reparameterization 가능

**답 작성 포인트**: “가능함” + $b = ae$ 재표현 + $e$는 $a$와 무관하게 샘플 → $\partial L/\partial a$ 경로 존재

#### Q2 — Graphical model 확률 관계식

**정답**: (1), (2), (5)

| 보기 | 판별 | 이유 |
|------|------|------|
| (1) $P(A,B,C,D,E) = P(A)P(B\|A)P(C\|A,B)P(D\|B)P(E\|C,D)$ | ✅ | DAG에 맞는 **chain rule factorization** |
| (2) $P(A,E\|B,C) = P(A\|B,C)P(E\|B,C)$ | ✅ | $B,C$가 주어지면 $A \perp E$ (그래프 구조상) |
| (3) $P(A,D\|E) = P(A\|E)P(D\|E)$ | ❌ | $E$만으로는 $A,D$ **조건부 독립 아님** ($B,C$ 경로 존재) |
| (4) $P(D,E\|B,C) = P(D\|B,C)P(E\|B,C)$ | ❌ | $D,E$는 $C$를 통해 연결 — $B,C$만으로 독립 아님 |
| (5) $P(A,D\|B,C,E) = P(A\|B,C,E)P(D\|B,C,E)$ | ✅ | $B,C,E$가 주어지면 $A \perp D$ |

**시험 연결**: 오토리그레시브 factorization $P(X_1,\ldots,X_T) = \prod_t P(X_t|X_{<t})$ 의 그래프 버전. 조건부 독립 판별 능력이 생성 모델 확률 이해의 기초.

#### Q3 — Diffusion ↔ Hierarchical VAE

**(a) 구조 대응**

| Diffusion | Hierarchical VAE | 학습 여부 |
|-----------|------------------|-----------|
| Forward $q(x_t|x_{t-1})$ (노이징) | Encoder (데이터→latent) | **고정** (하이퍼파라미터 스케줄) |
| Reverse $p_\theta(x_{t-1}|x_t)$ (디노이징) | Decoder (latent→데이터) | **학습** (신경망이 평균·분산 예측) |
| $x_T \approx \mathcal{N}(0,I)$ | Prior | 설계상 수렴 |

**(b) ELBO 3항**

1. **Reconstruction** — $x_0$ 복원
2. **Prior matching** — $x_T$가 표준 가우시안 (forward 설계상 **자동 만족**, 최적화에서 사라짐)
3. **Denoising matching** — $x_t$에서 $x_{t-1}$로 가는 분포 맞추기 → **reverse process 학습의 핵심**

## 문제 1 (확정)

다음과 같은 신경망을 고려해보자:

$$a = \text{ReLU}(Wx)$$
$$b \sim \text{Unif}(-a, a)$$
$$y = W'b$$

`Unif(-a,a)`는 $-a$와 $a$ 사이 구간의 균일분포이며, `~`는 해당 분포에서 샘플링했음을 의미한다. 이 신경망은 입력 $x$를 받아 첫 번째 layer를 거쳐 scalar $a$로 변환하고, $a$에 의해 정의된 구간의 균일분포로부터 $b$를 샘플링한 뒤, 선형 layer를 거쳐 출력 $y$를 만든다. 이 때 이 신경망이 역전파 알고리즘을 통해 $W$를 업데이트할 수 있는지 여부와 그 이유를 기술하시오.

### 답안

**가능함.**

$b$를 $b = a \cdot e$로 표현할 수 있고, 이때 $e$가 $\text{Unif}(-1, 1)$에서 샘플링되는 구조로 바꾸면 (**reparameterization trick**), $b$를 통한 loss의 gradient가 $a$로 전달되고, $a$를 통해 $W$로 역전파가 가능하다. 샘플링 자체는 미분 불가능하지만, **확률 변수를 고정 분포에서 뽑은 noise $e$와 학습 가능한 스케일 $a$의 곱**으로 분리하면 gradient path가 생긴다.

## 문제 3 (확정)

Diffusion model과 관련된 다음 두 질문에 답하시오.

**(a)** Diffusion이 hierarchical VAE와 어떻게 대응되는지 설명하시오. 특히 forward(noising)와 reverse(denoising) 과정이 각각 VAE의 어떤 요소(encoder/decoder)에 해당하며, 둘 중 무엇이 학습 가능하고 무엇이 고정인지 밝히시오.

**(b)** Diffusion의 ELBO는 세 항으로 정의된다. 세 항의 역할을 쓰고, reverse process를 학습시키는 핵심 항이 무엇인지 쓰시오.

### 답안

**(a)**

- **Forward** $q(x_t|x_{t-1})$: 데이터를 점진적으로 노이즈로 보내는 **encoder**에 해당한다. 학습 파라미터가 없는 **고정** 과정이며, prior(standard normal)로 수렴하도록 설계된다.
- **Reverse** $p_\theta(x_{t-1}|x_t)$: 노이즈를 점진적으로 복원하는 **decoder**에 해당한다. 가우시안의 평균·분산을 신경망으로 예측하는 **학습 가능**한 부분이다.
- 다수의 latent가 사슬처럼 연결된 점에서 **hierarchical VAE** 구조와 같다.

**(b)**

1. **Reconstruction** — 원본 $x_0$ 복원
2. **Prior matching** — $x_T$가 표준 가우시안에 가깝도록 (forward 설계상 자동)
3. **Denoising matching** — $x_t$에서 $x_{t-1}$로 가는 분포 맞추기

**Reverse process를 실제로 학습시키는 핵심 항은 denoising matching**이다.

# 3부. 남은 1문제 — 출제 예상 개념 정리

강의 「시험 출제 포인트」 기준, 확정 3문제가 담지 못한 영역을 **우선순위 순**으로 정리한다.

---

## ★★★ ① RNN 병목 + Attention + Transformer

**출제 포인트**: B2, B3, B4, B5, B6

| 개념 | 핵심 |
|------|------|
| **RNN 병목 ①** | 입력 전체 → **고정 크기 벡터 $C$** 하나로 압축 (치트시트 비유) → 정보 손실 |
| **RNN 병목 ②** | $h_t$가 $h_{t-1}$에 의존 → 학습·추론 **직렬** → 병렬화 불가 |
| **Attention** | 디코딩 시 **모든 인코더 시점에 직접 접근**; $\alpha_{it}$ = softmax 확률 가중합 |
| **Attention + gradient** | **Skip connection** 역할 → long-term modeling·optimization에 유리 |
| **Transformer** | recurrence **제거**; **학습**은 teacher forcing으로 **병렬** / **추론(생성)** 은 여전히 **직렬** |

**왜 4번째 문제에 가장 유력한가**
- 260622 오전 **핵심 주제**이나 확정 3문제 어디에도 없음 (Q2가 확률만, Q1·Q3가 VAE/Diffusion).
- 교수님이 ⭐로 반복 강조한 **“두 병목 → Attention → Transformer”** 흐름.
- 확정문제와 동일한 **“문제 제시 → 원인 → 해결 구조”** 서술형으로 내기 좋음.

---

## ★★★ ② VAE vs GAN — blur vs mode collapse

**출제 포인트**: C4, C5

| | GAN | VAE |
|---|-----|-----|
| 학습 안정성 | 불안정 (min-max, 로스 지그재그) | **ELBO**로 안정 |
| 모드 커버리지 | **mode collapse** (일부 모드만) | KL 페널티 → 모든 모드에 조금씩 확률 |
| 생성 품질 | **sharp** | **blur** (모드 사이 평균 효과) |
| 확률 해석 | 암시적 | **명시적** (가우시안, ELBO 분해) |

**VAE blur 원인**: multimodal $P(X)$에서 KL이 **모든 곳에 확률을 조금씩** assign → 인접 모드 **평균** → 뿌연 이미지.

**왜 유력한가**: Q1이 reparam만 다룸. VAE의 **생성 품질·학습 특성** 비교는 강의 표 전체가 시험 답안 그대로 — 전형적인 **2모델 비교형**.

---

## ★★☆ ③ 분류 vs 생성 + Likelihood maximization

**출제 포인트**: A1, A2, A3, A4

| 개념 | 핵심 |
|------|------|
| **생성의 정의** | 데이터 분포 $P(X)$ **추정** + **샘플링 = 생성** |
| **vs 분류** | 같은 cross-entropy 꼴이지만, 분류는 GT 확률(원핫)을 **알고**, 생성은 $P^*(X)$를 **모름** |
| **적분 불가** | $P(X)$에 대한 적분 직접 계산 불가 → **Monte Carlo** 샘플 평균으로 근사 |
| **공통 목표** | **Likelihood 최대화** (= NLL 최소화); 데이터 확률 ↑ → 비데이터 영역 ↓ |

**왜 유력한가**: 강의 **1부 서두** 핵심이나 확정에서 미출제. Q2(그래프)와 자연스럽게 이어지는 **확률 기초** 심화.

---

## ★★☆ ④ Autoregressive factorization + Transformer 병렬화

**출제 포인트**: A5, B1, B6

- $P(X_1,\ldots,X_T) = \prod_t P(X_t|X_{<t})$ — **chain rule**; RNN next-word = 생성 목적함수
- Factorization **순서**는 joint 동일하나 각 항의 **분산(불확실성)** 은 다름
- Transformer **학습**: teacher forcing으로 정답 토큰 미리 넣어 **병렬** / **추론**: 한 토큰씩 **직렬** (KV caching으로 비용 완화)

**왜 중간 우선순위인가**: ①번과 주제 겹침. 단독으로 “AR + teacher forcing”만 물을 수도 있음.

---

## ★★☆ ⑤ Noise prediction + Diffusion vs VAE 차원

**출제 포인트**: D5, D6

- 학습 목표: $X_t$와 $t$가 주어졌을 때 섞인 노이즈 **$\epsilon$ 예측** (DDPM 표준)
- VAE: **저차원** latent $Z$ | Diffusion: 매 단계 latent = **풀 해상도 텐서** (픽셀 공간 노이징)

**왜 중간 우선순위인가**: Q3가 hierarchical VAE·ELBO를 이미 다뤄 **부분 중복**. 4번째가 diffusion **심화**(noise pred)일 가능성은 있으나, 260622 미출제가 더 큼.

---

## ★☆☆ ⑥ Flow Matching

**출제 포인트**: E1, E2, E3

- 분포 간 **velocity field** 회귀; prior 가우시안 **강제 아님** (diffusion보다 general)
- Score · diffusion · flow = **같은 패밀리** (연속 분포 변환)

**왜 낮은가**: 강의에서 **맛보기** 수준. 확정 3문제가 이미 diffusion을 깊게 다룸.

## 예상문제 1 — RNN 병목, Attention, Transformer (★ 최우선)

Sequence-to-sequence 기계 번역을 위한 초기 encoder-decoder RNN 구조의 한계 두 가지를 설명하고, Attention이 각각을 어떻게 완화하는지 서술하시오. 이어서 Transformer가 recurrence를 제거한 뒤, **학습**과 **추론(생성)** 에서 병렬화 가능 여부가 어떻게 달라지는지 설명하시오.

### 모범 답안

**RNN encoder-decoder의 한계 두 가지**

1. **정보 병목 (long-term dependency)**  
   입력 문장 전체를 **고정 크기 벡터 $C$** 하나에 압축해야 한다. 문장이 길수록 정보 손실이 커지고, 디코딩 시 입력의 세부 내용을 복원하기 어렵다.

2. **병렬화 불가**  
   $h_t$ 계산에 $h_{t-1}$이 필요해 학습·추론 모두 **시간 축으로 직렬** 처리해야 한다.

**Attention의 완화**

- 디코딩할 때 **모든 인코더 타임스텝의 hidden state에 직접 접근**한다. 고정 벡터 $C$ 하나에 의존하지 않고, 매 출력 시점마다 입력 전체에서 **관련 있는 부분을 동적으로 선택**한다.
- Attention weight $\alpha_{it}$는 softmax로 정규화된 **확률 가중치**이며, query(현재 디코더 상태)와 key(인코더 상태)의 유사도로 계산된다.
- Attention 경로는 **skip connection**처럼 gradient가 멀리까지 흘러, long-term dependency와 optimization 모두에 유리하다.

**Transformer의 병렬화**

- Self-attention만으로 과거 정보를 볼 수 있으므로 **재귀 연결(recurrence)을 제거**한다. 토큰 간 attention은 **병렬** 계산 가능하다.
- **학습**: teacher forcing으로 정답 토큰을 미리 넣어 모든 위치의 loss를 **동시에(병렬)** 계산할 수 있다.
- **추론(생성)**: 오토리그레시브 conditioning 때문에 여전히 **한 토큰씩 직렬** 생성한다. KV caching 등으로 비용을 줄일 수 있으나, 본질적으로 순차 생성이다.

## 예상문제 2 — VAE vs GAN 비교

VAE와 GAN의 학습 안정성, 모드 커버리지(mode coverage), 생성 품질, 확률 해석 측면에서 차이를 비교하시오. 특히 VAE 생성 결과가 **뿌옇게(blur)** 나오는 이유를 multimodal 데이터 분포와 KL penalty 관점에서 설명하시오.

### 모범 답안

| 측면 | GAN | VAE |
|------|-----|-----|
| **학습 안정성** | Min-max 게임, loss가 지그재그 → **불안정** | **ELBO** 최대화로 **안정적** |
| **모드 커버리지** | Mode collapse — 일부 모드만 생성해도 판별기를 속일 수 있어 **페널티 없음** | KL이 0 확률 영역에 **큰 페널티** → 모든 모드에 조금씩 확률 assign |
| **생성 품질** | **Sharp** (특정 모드에 집중) | **Blur** (모드 사이 평균) |
| **확률 해석** | **암시적** (명시적 likelihood 없음) | **명시적** (가우시안 가정, ELBO = reconstruction + KL) |

**VAE blur 원인**

- 실제 이미지 분포 $P(X)$는 **multimodal** (같은 클래스라도 다양한 외형).
- VAE의 KL 항은 posterior가 prior($\mathcal{N}(0,I)$)에서 너무 멀어지지 않게 하므로, decoder가 한 모드에 확률 질량을 **집중**하기 어렵다.
- 결과적으로 인접한 여러 모드 사이를 **평균** 내는 형태의 출력이 나와 **뿌연 이미지**가 된다.
- 반면 GAN은 특정 모드만 잘 생성해도 되므로 sharp하지만, **mode collapse**로 다양성이 떨어질 수 있다.

## 예상문제 3 — 분류 vs 생성 + Likelihood Maximization

생성 모델에서 “생성”이 무엇을 의미하는지 설명하고, 이미지 분류와 비교했을 때 목적 함수(cross-entropy)의 **공통점**과 **차이점**을 서술하시오. 생성 모델이 $P^*(X)$를 직접 알지 못하는 상황에서 어떻게 likelihood를 최대화하는지 **Monte Carlo** 관점에서 설명하시오.

### 모범 답안

**생성의 정의**

- 생성 모델은 데이터가 따르는 **확률 분포 $P(X)$를 추정**하고, 그 분포에서 **샘플링**하여 새 데이터를 만든다. **샘플링 = 생성**이다.

**분류 vs 생성 — 목적 함수**

| | 분류 | 생성 |
|---|------|------|
| **Cross-entropy 꼴** | 동일하게 사용 | 동일하게 사용 |
| **GT 확률** | 원핫 등 **알고 있음** | $P^*(X)$를 **아무도 모름** |
| **적분** | (클래스 수 finite) 계산 가능 | $P(X)$에 대한 **적분 불가** (고차원) |
| **최종 목표** | 정답 클래스 확률 ↑ | **Likelihood $P_\theta(X)$ ↑** (= NLL ↓) |

**Monte Carlo로 likelihood 최대화**

- 관측 데이터 $X_i$는 이미 $P^*$에서 샘플된 것이므로, $P^*$를 직접 알 필요가 없다.
- intractable한 적분 $\mathbb{E}_{P^*}[\log P_\theta(X)]$ 대신, 유한 샘플의 평균 $\frac{1}{N}\sum_i \log P_\theta(X_i)$로 근사한다.
- 이를 최대화하면 **negative log-likelihood minimization**이 되며, 학습 데이터에 높은 확률을 부여하고 나머지 영역 확률은 낮아져 **노이즈·비데이터를 구분**하게 된다.

# 5부. 시험 직전 체크리스트

### 확정 3문제 (반드시 서술 연습)

- [ ] Reparameterization: $b = ae$, $e \sim \text{Unif}(-1,1)$ / 샘플링 미분 불가 → 재매개변수화
- [ ] Graphical model: (1)(2)(5) 정답 / 조건부 독립 = 그래프 구조 판별
- [ ] Diffusion forward = **고정** encoder / reverse = **학습** decoder
- [ ] ELBO 3항: Reconstruction / Prior matching / **Denoising matching**(핵심)

### 예상 4번째 (우선 대비)

- [ ] RNN 병목: **고정 벡터** + **직렬 의존**
- [ ] Attention: 인코더 전 시점 **직접 참조** + skip connection
- [ ] Transformer: 학습 **병렬**(teacher forcing) / 추론 **직렬**
- [ ] VAE blur = KL → 모드 사이 **평균** / GAN = **mode collapse**
- [ ] 생성 = 분포 추정 + 샘플링 / Monte Carlo → **likelihood 최대화**

### 보조 대비

- [ ] Chain rule: $\prod_t P(X_t|X_{<t})$ — AR factorization
- [ ] Noise prediction: $X_t, t$ → **$\epsilon$** 예측
- [ ] Diffusion latent = **풀 해상도** / VAE latent = **저차원**
- [ ] Beta-VAE: $\beta > 1$ on KL → disentanglement
- [ ] Flow matching: **velocity field** 회귀, prior 제약 느슨
