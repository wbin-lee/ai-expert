## 1.5 확정문제별 심층 분석

### Q1 — Linear Attention (3소문항)

**시험이 검증하는 것**
- Softmax 없는 attention을 **상태 행렬 \(S\)** 로 쓸 수 있는가  
- \(S\)를 **연관 메모리**로 보고 **MSE online 학습**이 delta update가 됨을 아는가  

**(a) Re-arrange + recurrent**

\[
o_t = \sum_{j=1}^{t} (q_t^\top k_j)\, v_j
\;\;\Longrightarrow\;\;
o_t = S_t q_t
\]

| 항목 | 정답 |
|------|------|
| \(S_t\) | \(\displaystyle S_t = \sum_{j=1}^{t} v_j k_j^\top\) |
| Recurrent | \(\displaystyle S_t = S_{t-1} + v_t k_t^\top\) |

**강의 연결**: associativity로 \(Q(K^\top V)\) 꼴 → 유한 \(D\times D\) 상태. Linear RNN.

**(b) Associative memory loss**

\(f_S(k)=Sk\)가 키로 밸류를 예측. Loss \(\frac{1}{2}\|a-b\|^2\) 이면

\[
L_t(S) = \frac{1}{2} \big\| S k_t - v_t \big\|^2
\]

**(c) Online GD (Delta rule)**

\[
S_t = S_{t-1} - \beta_t \nabla L_t(S_{t-1})
= S_{t-1} - \beta_t \big(S_{t-1} k_t - v_t\big) k_t^\top
\]

**시험 팁**: (a)만 외우면 안 된다. 교수님 출제 발언의 “업데이트 룰”은 **(a)의 단순 합**뿐 아니라 **(c)의 에러 보정 write**까지 포함한 강의 클라이맥스다.  
\(\nabla_S \frac{1}{2}\|Sk-v\|^2 = (Sk-v)k^\top\) 한 줄만 기억하면 (c)가 나온다.

---

### Q2 — ERM과 i.i.d.

**시험이 검증하는 것**
- ERM이 무엇을 줄이는지 (**평균** risk)  
- 그 정당성이 **i.i.d.** 가정에 기대며, 가정이 깨지면(분포 shift, 그룹 불균형, spurious) 실패  

| 빈칸 | 정답 |
|------|------|
| (A) | **평균** (average of losses) |
| (B) | 학습 데이터가 **i.i.d.** 분포를 따른다는 가정 |

**강의 연결**: Fairness·Group DRO로 넘어가기 위한 포석. “평균만 보면 다수 그룹에 맞춰지고 worst group이 망한다”의 전제.

**시험 팁**: (B)는 “i.i.d.가 아닐 때”를 쓰는 문제 문장에 맞춰, 답은 **“i.i.d. 가정”** 자체. 예: domain shift, group-wise non-i.i.d.를 한 줄 보강하면 가산점.

---

### Q3 — InfoNCE 완성

**시험이 검증하는 것**
- InfoNCE = **positive를 분류하는 softmax**  
- temperature \(\tau\), positive 1개 + negative \(M\)개  

\[
\mathcal{L}_{\mathrm{InfoNCE}}(f)
= \mathbb{E}\!\left[
-\log \frac{(A)}{(B)}
\right]
\]

| | 정답 |
|--|------|
| **(A)** | \(\exp\!\big(f(x)^\top f(x^+)/\tau\big)\) |
| **(B)** | \(\exp\!\big(f(x)^\top f(x^+)/\tau\big) + \sum_{i=1}^{M}\exp\!\big(f(x)^\top f(x_i^-)/\tau\big)\) |

**직관**: (A)/(B) = “anchor feature가 positive를 고를 확률”. minimize NLL → pos 유사도↑, neg↓.  
SimCLR·MoCo 모두 이 loss 계열(negative 공급 방식만 다름).

**시험 팁**: \(\tau\) 위치, **분자=positive만**, 분모=pos+모든 neg. \(f(x)^\top f(\cdot)\) 내적(또는 sim) 형태를 그대로 쓸 것.

## 문제 1 (확정)

Linear attention에서 timestep \(t\)에서 query \(q_t\)에 대한 output \(o_t\)의 수식은 다음과 같다.

\[
o_t = \sum_{j=1}^{t} \big(q_t^\top k_j\big)\, v_j
\]

### (a)

이 수식을 re-arrange하여 \(o_t = S_t q_t\) 형태로 나타냈을 때, \(S_t\)의 식을 구하고 recurrent update 식을 구하시오.

#### 답안

\[
S_t = \sum_{j=1}^{t} v_j k_j^\top,\qquad
S_t = S_{t-1} + v_t k_t^\top
\]

### (b)

\(S_t\)를 key-value를 저장하는 associative neural memory로 해석하는 경우, \(f_S(k)=Sk\)로 정의하고  
\(\mathrm{Loss}(a,b)=\frac{1}{2}\|a-b\|^2\)로 정의하였을 때, time step \(t\)에서의 loss function \(L_t(S)\)를 쓰시오.

#### 답안

\[
L_t(S) = \frac{1}{2} \big\| S k_t - v_t \big\|^2
\]

### (c)

해당 loss function에 기반하는 online gradient descent 업데이트 식을 쓰시오.  
(\(S_{t-1}\)이 주어지고 \(k_t,v_t\)가 들어왔을 때, learning rate \(\beta_t\)를 사용하여 \(S_t\)를 업데이트)

#### 답안

\[
S_t = S_{t-1} - \beta_t \nabla L_t(S_{t-1})
= S_{t-1} - \beta_t \big(S_{t-1} k_t - v_t\big) k_t^\top
\]

## 문제 2 (확정)

ERM (Empirical Risk Minimization)은 각 학습데이터에서 계산하는 loss 값들의 **((A))** 을 최소화하는 방식으로 학습을 진행한다. 그런데, 이 방식은 데이터가 어떤 가정 **((B))** 을 따르지 않을 때 잘 동작하지 않는지 쓰시오.

#### 답안

- **(A)**: **평균**  
- **(B)**: 학습데이터가 **i.i.d.** 분포를 따른다는 가정  

---

## 문제 3 (확정)

SimCLR, MoCo 등 contrastive learning 기반 self-supervised learning에서 널리 쓰이는 typical한 InfoNCE loss function의 꼴을 (A)와 (B)에 들어갈 수식을 채워 완성하시오.

**Notation**  
- \(x\): anchor, \(x^+\): positive, \(x_i^-\): negative (\(i=1,\ldots,M\))  
- \(f(x)\): feature extractor  

\[
\mathcal{L}_{\mathrm{InfoNCE}}(f)
= \mathbb{E}_{(x,x^+)\sim p_{\mathrm{pos}},\,\{x_i^-\}\sim p_{\mathrm{data}}}
\left[
-\log \frac{(A)}{(B)}
\right]
\]

#### 답안

\[
(A) = \exp\!\left(\frac{f(x)^\top f(x^+)}{\tau}\right)
\]

\[
(B) = \exp\!\left(\frac{f(x)^\top f(x^+)}{\tau}\right)
+ \sum_{i=1}^{M}
\exp\!\left(\frac{f(x)^\top f(x_i^-)}{\tau}\right)
\]

## ★★★ ① Fairness Metric 정의 — DP / EOpp / EO

**출제 포인트**: 교수님 직접 언급 **「Fairness 매트릭 정의」** — 확정 PDF에 **유일하게 없는** 명시 항목  

**왜 4번째 최유력인가**
- 출제 발언 4축 중 PDF 미반영 = 이 주제뿐  
- Q2가 ERM 입구만 물어 **fairness 본론이 비어 있음**  
- 확정 스타일(식·짧은 정의)과 잘 맞음: 세 지표를 **조건부 확률 식**으로 쓰게 하기 좋음  

| 지표 | 정의 (이진 \(\hat{Y}, A, Y\)) | \(Y\) 조건 | 한 줄 |
|------|------------------------------|------------|------|
| **DP** (Demographic Parity) | \(P(\hat{Y}{=}1\mid A{=}0)=P(\hat{Y}{=}1\mid A{=}1)\) | **없음** | 그룹별 **긍정 예측률** 동일 |
| **EOpp** (Equal Opportunity) | \(P(\hat{Y}{=}1\mid Y{=}1,A{=}0)=P(\hat{Y}{=}1\mid Y{=}1,A{=}1)\) | **\(Y=1\)만** | 참양성(기회) 동일 |
| **EO** (Equalized Odds) | \(Y=0\)·\(Y=1\) 모두에서 \(\hat{Y}\) 분포 일치 (TPR·FPR 등) | **\(Y=0,1\)** | 더 강한 그룹 공정 |

**자주 묻는 함정**
- DP는 base rate(\(P(Y{\mid}A)\))가 다를 때 예측률을 억지로 맞추면 **역차별** 논쟁  
- EOpp/EO는 “자격이 있는(\(Y=1\)) / 라벨 조건” 관점  

---

## ★★☆ ② Group DRO vs ERM (Q2의 자연 연장)

**출제 포인트**: 2일차 fairness 본론, Q2와 쌍  

| | ERM | Group DRO |
|--|-----|-----------|
| 목표 | 전체 loss **평균** ↓ | 그룹 loss 중 **worst** 쪽을 집중 개선 |
| 실패 모드 | 다수 그룹에 맞춰 **worst-group** 붕괴 | overall이 약간 줄 수 있음 |
| 직관 | i.i.d.·평균 최적 | non-i.i.d. 그룹, spurious 완화 |

Q2가 이미 ERM을 물었으므로, 4번째가 “ERM의 한계를 넘는 방법”으로 Group DRO를 물을 수 있다. 다만 교수님 발언에 **metric 정의**가 더 명시적이라 **1순위보다는 아래**.

## 예상문제 1 — Fairness Metrics (★ 최우선)

그룹 공정성(group fairness)에서 널리 쓰이는 세 지표에 대해 답하시오.  
(민감 속성 \(A\), 정답 라벨 \(Y\), 모델 예측 \(\hat{Y}\)는 모두 이진이라고 가정)

### (a)
Demographic Parity (DP), Equal Opportunity (EOpp), Equalized Odds (EO)를 각각 **조건부 확률 식**으로 쓰시오.

### (b)
세 지표를 **정답 라벨 \(Y\)를 조건에 넣는지 여부** 기준으로 한 줄씩 구분하여 설명하시오.

### (c) (선택 보강)
DP만을 강하게 강제할 때 생길 수 있는 한계를 한 가지 쓰시오.

### 모범 답안

**(a)**

- **DP**: \(P(\hat{Y}=1\mid A=0) = P(\hat{Y}=1\mid A=1)\)  
- **EOpp**: \(P(\hat{Y}=1\mid Y=1, A=0) = P(\hat{Y}=1\mid Y=1, A=1)\)  
- **EO**: \(Y=0\)과 \(Y=1\) **모두**에서  
  \(P(\hat{Y}=1\mid Y=y, A=0) = P(\hat{Y}=1\mid Y=y, A=1)\) (\(y\in\{0,1\}\))  
  (동치 표현: 그룹 간 TPR·FPR이 모두 같음)

**(b)**

| 지표 | \(Y\) 조건 |
|------|------------|
| DP | **넣지 않음** — 그룹별 긍정 **예측률**만 맞춤 |
| EOpp | **\(Y=1\)만** — “자격이 있는” 집단 안 기회(TPR) |
| EO | **\(Y=0,1\) 모두** — 더 강한 조건 |

**(c)**  
실제 base rate \(P(Y{\mid}A)\)가 그룹마다 다른데도 예측 긍정률을 같게 강제하면, 자격이 다른 집단에 동일 비율 할당이 되어 **역차별·비현실적 제약**이 될 수 있다.

## 예상문제 2 — Group DRO (ERM 대비)

### (a)
ERM이 최소화하는 대상과 Group DRO가 관심을 두는 대상을 각각 한 줄로 쓰시오.

### (b)
Spurious correlation(예: Waterbirds에서 배경–라벨 상관)이 있을 때, ERM 학습 모델에서 흔히 관찰되는 실패 패턴을 쓰시오.

### (c)
Group DRO가 overall accuracy와 worst-group accuracy에 전형적으로 미치는 영향을 쓰시오.

### 모범 답안

**(a)**  
- **ERM**: 전체 샘플 loss의 **평균(empirical risk)** 최소화.  
- **Group DRO**: 그룹별 risk 중 **worst(가장 나쁜 그룹)** 쪽을 줄이도록 학습(그룹 weight \(q_g\)로 어려운 그룹에 무게).

**(b)**  
다수 조합(예: 물새–물 배경)만으로도 평균 loss가 줄어 **배경 등 spurious feature에 의존**. 소수 그룹(물새–숲 등)에서 성능이 **크게 붕괴**(worst-group accuracy 저하).

**(c)**  
Overall accuracy는 소폭 감소할 수 있으나, **worst-group accuracy는 상승**하고 그룹 간 격차가 줄어드는 경우가 많다. (Q2의 “i.i.d./평균 ERM 한계”에 대한 알고리즘적 대응.)

## 예상문제 3 — SimCLR vs MoCo (InfoNCE 공유 전제)

SimCLR과 MoCo는 모두 InfoNCE 계열 contrastive loss를 사용한다. 다음 빈칸을 채우시오.

### (a)
Unlabeled 이미지 SSL에서 전형적인 **positive pair**는 어떻게 만드는가? (한 줄)

### (b)
| | SimCLR | MoCo |
|--|--------|------|
| Negative를 주로 어디서 얻는가? | **(A)** | **(B)** |
| Key/negative encoder 특징 | 공유 encoder end-to-end | **(C)** |

### (c)
SimCLR에서 batch size가 커질수록 성능이 좋아지기 쉬운 이유를 InfoNCE 관점에서 한 줄로 쓰시오.

### 모범 답안

**(a)**  
같은 이미지에 **서로 다른 random augmentation**을 두 번 적용해 얻은 두 view를 positive pair로 둔다.

**(b)**  
- **(A)**: 같은 **mini-batch** 안의 다른 샘플(의 feature)  
- **(B)**: **Queue**에 쌓인 과거 배치의 key feature (FIFO)  
- **(C)**: **Momentum (EMA) encoder**로 key를 추출 (query encoder만 backprop)

**(c)**  
Batch가 클수록 InfoNCE 분모의 **negative 수·다양성**이 커져 **hard negative**가 포함될 확률이 높아지고, 변별력 있는 representation 학습에 유리하다.

# 5부. 시험 직전 체크리스트

### 확정 3문제 (반드시 식 쓰기 연습)

- [ ] \(o_t=\sum(q_t^\top k_j)v_j\) → \(S_t=\sum v_j k_j^\top\), \(S_t=S_{t-1}+v_t k_t^\top\), \(o_t=S_t q_t\)
- [ ] \(L_t=\frac{1}{2}\|Sk_t-v_t\|^2\)
- [ ] \(S_t=S_{t-1}-\beta_t(S_{t-1}k_t-v_t)k_t^\top\)  (**Delta / online GD**)
- [ ] ERM = loss의 **평균** / 깨지면 안 되는 가정 = **i.i.d.**
- [ ] InfoNCE:  
  \(\displaystyle -\log\frac{\exp(f(x)^\top f(x^+)/\tau)}{\exp(f(x)^\top f(x^+)/\tau)+\sum_i\exp(f(x)^\top f(x_i^-)/\tau)}\)

### 예상 4번째 (우선 대비)

- [ ] **DP**: \(P(\hat{Y}{\mid}A)\) 동일, **\(Y\) 없음**
- [ ] **EOpp**: \(P(\hat{Y}{\mid}Y{=}1,A)\) 동일
- [ ] **EO**: \(Y=0,1\) 모두 조건
- [ ] DP 한계: base rate 차이 → 역차별 논쟁
- [ ] (보조) Group DRO = worst-group, ERM = average

### 보조 대비

- [ ] positive = augmentation view; temperature \(\tau\)
- [ ] SimCLR = 큰 배치 / MoCo = queue + momentum
- [ ] Decay \(\gamma\): recency bias / vanilla linear의 약점
- [ ] Full attention: exact KV, \(O(L^2)\) · Linear: 유한 \(S\), 간섭·용량 \(D\)

### 답안 작성 팁 (확정 PDF 스타일)

1. **식을 먼저** 쓰고, 한 줄 해석은 여유 있을 때만  
2. 벡터/전치 방향: \(v k^\top\) (rank-1), \(o=Sq\) 일관 유지  
3. InfoNCE는 **분자=pos만**, 분모=pos+neg — 순서 실수 금지  
4. Fairness는 **\(Y\)를 조건에 넣는지**를 표로 쓰면 감점 방지  
5. 장문 에세이보다 **빈칸·정의·한 줄 식**이 본 시험 톤
