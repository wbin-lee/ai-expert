# 260410 오후 강의 요약 — 최적화(Optimization) · 정규화(Regularization)

## 1. 최적화(Optimization)

### 1.1 기본 원리
```
w ← w − η · ∇E(w)
```
- 파라미터 w를 손실함수 E의 음의 그래디언트 방향으로 이동
- η : Learning Rate (스텝 크기)

### 1.2 Optimizer의 진화

#### ① Vanilla SGD
```
dw = -η · ∇E
```
- 단순하지만 그래디언트 노이즈가 많음

#### ② Momentum
```
v_t = β v_{t-1} + (1-β) · ∇E      (EWMA)
w  ← w − η · v_t
```
- 과거 그래디언트의 **지수가중 이동평균** → 관성으로 노이즈 완화

#### ③ RMSProp
```
v_t = β · v_{t-1} + (1-β) · (∇E)²
η_adapt = η / √(v_t + ε)
w ← w − η_adapt · ∇E
```
- 가파른 방향은 작은 학습률, 평평한 방향은 큰 학습률

#### ④ Adam (현대 표준)
```
m_t = β₁ · m_{t-1} + (1-β₁) · ∇E       (1차 모멘트 = 속도)
v_t = β₂ · v_{t-1} + (1-β₂) · (∇E)²    (2차 모멘트 = 가속도)
ŵ   = w − η · m_t / √(v_t + ε)
```
- Momentum + RMSProp 결합
- 기본 β₁=0.9, β₂=0.999, ε=1e-8

#### ⑤ 변형
- **AdamW** : Weight Decay를 L2 정규화와 분리 (현대 LLM 표준)
- **RAdam** : Warm-up 자동화

### 1.3 Learning Rate 스케줄
- 초반 : 큰 η (빠른 수렴)
- 중반 이후 : 점진적 decay
- 대배치 트레이닝 : **Warm-up** (초반 점진적 증가)
- 일반 : Step/Cosine/Exponential Decay

## 2. 정규화(Regularization)

### 2.1 기본 아이디어
**오버피팅 방지·일반화 향상.** 모델 복잡도에 패널티 추가:
```
E_aug = E_train + λ · Ω(w)
```
- Ω(w) : 복잡도 측정
- λ : 정규화 강도

### 2.2 노이즈 관점
지구상 대부분 신호는 노이즈 포함. 노이즈 = 고주파 성분. **정규화 = 노이즈 피팅 방지** (저주파 패스 필터 역할).

## 3. L1 / L2 / Elastic Net

### 3.1 L1 Regularization (Lasso)
```
Ω(w) = Σ |wᵢ|
```
- 기하학적 제약 : 다이아몬드 영역
- 최적점이 꼭짓점에서 만남 → **많은 wᵢ = 0** (sparse)
- 변수 선택 효과

### 3.2 L2 Regularization (Ridge)
```
Ω(w) = Σ wᵢ²
```
- 기하학적 제약 : 원형 영역
- 최적점이 원 위 → **모든 wᵢ 생존**, 크기만 축소(shrinkage)

### 3.3 Elastic Net
```
Ω(w) = α·L1 + (1-α)·L2
```
- α=1 : Lasso / α=0 : Ridge
- L1·L2 장점 결합

## 4. Dropout

### 4.1 원리
훈련 중 뉴런을 확률 p(보통 0.5) 로 "끔" (output = 0).
- 매 미니배치마다 **랜덤한 서브 네트워크** 학습
- 테스트 시엔 모든 뉴런 사용 (또는 가중 평균)

### 4.2 효과
- **암묵적 앙상블** : 서로 다른 서브넷의 평균
- 뉴런 하나에 과도 의존 방지 → 공적응(co-adaptation) 완화
- "백신 효과" : 약한 노이즈 노출로 강인성 획득

### 4.3 제안자 및 생물학적 해석
Geoffrey Hinton (2012). 뉴런의 일부를 끔으로써 뇌의 잡음 적응과 유사한 효과.

## 5. Early Stopping

### 5.1 원리
훈련 중 **Validation Error** 모니터링 → 최소점에서 중지.
- 장점 : 구현 간단, 효과 큼
- 실전 : Val Error 증가 추세가 명확해질 때 중지

### 5.2 근거
`Dev Error ≈ Generalization Error` 라는 근사.

## 6. Double Descent (현대적 관찰)

전통 이론은 U자(복잡도 ↑ → 일반화 ↓)를 예상했으나, 매우 복잡한 모델에서 **다시 일반화가 좋아지는** 현상이 관찰됨.
- 원인 (추정) : SGD 자체의 암묵적 정규화, 좋은 네트워크 구조의 귀납적 편향
- 거대 모델(LLM)도 잘 일반화하는 이유

## 7. 핵심 정리

1. 최적화 : `w ← w − η · ∇E`.
2. Momentum(EWMA)로 노이즈 제거.
3. RMSProp : 방향별 적응 학습률.
4. Adam = Momentum + RMSProp (현대 표준).
5. AdamW = weight decay 분리판.
6. LR Decay + Warm-up (대배치).
7. L1 = sparse (변수 선택), L2 = shrinkage (모두 축소).
8. Elastic Net = L1 · L2 혼합.
9. Dropout = 랜덤 뉴런 off → 암묵적 앙상블.
10. Early Stopping = Val Error 최소점 중지.
11. 정규화 = 노이즈 피팅 방지 = 일반화 향상.
12. λ는 휴리스틱·CV로 결정.
13. Double Descent로 거대 모델도 잘 일반화.
14. SGD 자체도 암묵적 정규화.
15. 현대 베스트 = 복잡하지만 정규화된 딥러닝.

### 강의 중 주요 직관·일화
- **노아의 방주** : L1은 대부분 죽이고(선택), L2는 모두 살리되 다이어트(축소).
- **약의 복용량** : λ 과하면 underfit, 부족하면 overfit.
- **주식 저점 타이밍** : Early Stopping을 주식 매매에 비유.
- **아이스하키 스틱 1000번 압축** : 정규화가 모델을 단련시키는 비유.
- **코로나 백신** : Dropout이 약한 노이즈로 면역(일반화) 형성.
