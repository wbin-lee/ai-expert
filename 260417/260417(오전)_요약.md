## 1. Generative vs Discriminative

### 1.1 판별형(Discriminative)
- 학습 대상 : `p(y|x)` — 결정 경계
- 장점 : 분류·회귀에 직접적, 효율적
- 한계 : 분포 전체 모름 → 샘플 생성 불가

### 1.2 생성형(Generative)
- 학습 대상 : `p(x)` 또는 `p(x, y)`
- 분포 전체를 학습 → **새 샘플 생성 가능**
- `p(y|x)` 는 베이즈 정리로 도출 가능 (`p(y|x) = p(x|y) p(y) / p(x)`)

## 2. 고차원 데이터의 샘플링 난제

### 2.1 예시
- 100×100 이미지 = 10,000 차원
- FHD 컬러 = 약 600만 차원
- 가능한 조합은 무한대에 가까움

### 2.2 문제
- **랜덤 샘플링** 시 대부분 무의미한 픽셀 덩어리
- 의미 있는 이미지는 **전체 공간의 극히 일부**
- 확률 `P(x)` 가 0이 아닌 샘플을 뽑는 것이 본질적으로 어려움

### 2.3 Infinite Monkey Theorem
무한 원숭이가 타이핑하면 셰익스피어 작품도 확률적으로 가능하지만, 유한 자원에서는 불가능 → Approximation 필연.

## 3. Manifold Hypothesis (다양체 가정)

### 3.1 정의
**고차원 데이터가 실제로는 저차원 다양체(manifold) 상에 존재한다.**
- Swiss Roll 예 : 3D 공간에 말린 2D 평면, 1D로 표현 가능
- 현대 딥러닝의 핵심 가정 : 이 저차원 구조를 발견·학습

### 3.2 직관 예시
- **이미지** : 한 사람의 얼굴 사진 → 색조·밝기 변경해도 여전히 같은 사람
- **음성** : "안녕하세요"의 속도·톤 조정 → 여전히 같은 의미
- **텍스트** : 의미 있는 문장들은 전체 문자열 공간의 극히 일부

### 3.3 비유
- 전체 방(공간) 안에 모니터(의미 있는 데이터)만 존재
- 시각 장애인이 코끼리 더듬기 : **한 점 찾기는 어렵지만, 찾은 후 주변 탐색은 쉬움** → 데이터가 연결된 저차원 구조에 몰려 있음

## 4. 파인만의 원칙과 학문의 단계

### 4.1 파인만 명언
> "I cannot create what I do not understand."

**생성 = 이해**의 본질적 연결. 생성할 수 있으면 이해한 것, 이해해야 제어 가능.

### 4.2 학문 진행 3단계
1. **Science (과학)** — 현상 이해 (고차원 데이터 구조)
2. **Engineering (공학)** — 제어·생산 (생성 가능)
3. **Business (사업)** — 이윤 창출

### 4.3 현재 AI의 위치
- 과거 : 생성형 AI가 필요하다고 과학자들이 주장했지만 외면
- 현재 : "판별형으로 충분한가?"라는 질문조차 나오지 않는 명백한 수요
- 미래 : 생성 능력의 정치·윤리적 파급력 고려 필요

## 5. 생성형 AI의 응용 분야

| 응용 | 설명 |
|------|------|
| **데이터 생성** | 부족한 학습 샘플 보충 (Data Augmentation) |
| **Imputation** | 결측치 채우기 |
| **Semi-supervised Learning** | 레이블 없는 데이터 활용 |
| **Generative Replay** | 연속학습에서 이전 지식 망각 방지 |
| **Super-Resolution** | 저해상도 → 고해상도 (삼성 TV 4K/8K) |
| **Image-to-Image Translation** | 스타일 변환, 도메인 적응 |
| **Missing Person Recovery** | 어린 시절 사진 → 현재 얼굴 추정 (중국 연 수십명 구조) |
| **Inverse Problems** | 과학·산업 역문제 해결 |

## 6. Deep Generative Model의 분류 예고

- **Explicit (Likelihood-based)** : Autoregressive(GPT), **VAE**, Flow, Diffusion
- **Implicit** : **GAN**

## 7. 핵심 정리

1. Generative는 `p(x)` 전체, Discriminative는 경계만.
2. 고차원 랜덤 샘플 대부분은 무의미 → 샘플링 난제.
3. Manifold Hypothesis : 데이터는 저차원 다양체에 몰려 있다.
4. 현대 딥러닝은 이 다양체 학습의 과정.
5. 파인만 원칙 : 이해 → 생성 → 제어.
6. Inverse Problem 해결이 산업적 수익 큼.
7. Super-Resolution = 대표 소비재 응용 (삼성 TV).
8. Generative Replay로 Catastrophic Forgetting 완화.
9. Missing Person Recovery 는 실제 사회적 구조 사례.
10. Imputation·Semi-supervised 에서 생성형이 강력.
11. Infinite Monkey Theorem → 유한 자원에선 Approximation 필수.
12. Swiss Roll 예시로 Manifold 직관 시각화.
13. Science → Engineering → Business 파이프라인.
14. 생성 능력의 윤리적·정치적 파급력 고려 필요.
15. Curse of Dimensionality ↔ Manifold 로 돌파.

### 강의 중 주요 직관·일화
- **오펜하이머 영화 일화** : 물리학과 교수들이 가족 무시하고 영화관 방문.
- **건물 화장실 스티커** : "오래된 지식을 가르치는 선생보다 슬픈 건 없다".
- **19세기 런던 추리소설 강좌** : "중국인 등장 금지" — 영리함으로 스토리 붕괴 우려. 현대 AI의 "Attention Mechanism" 이 그 "중국인".
- **구한말 신기술 오해** : 라디오=귀신, 전화=영혼 탈취 — 이해 없으면 기술은 마술.
- **시각 장애인 코끼리 더듬기** : MCMC vs Variational Inference 의 차이 비유.
