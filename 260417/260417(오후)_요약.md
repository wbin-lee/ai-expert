# 260417 오후 강의 요약 — Variational Inference · Autoencoder · ELBO

## 1. Posterior Inference의 어려움

### 1.1 문제 정의
관찰 x가 주어졌을 때 숨겨진 변수 z의 분포 `p(z|x)` 를 구하는 것 → 일반적으로 **Intractable**.

```
p(z|x) = p(x|z) p(z) / p(x)
p(x) = ∫ p(x|z) p(z) dz     ← 이 적분이 어려움
```

### 1.2 왜 어려운가
- 고차원 z에서 적분이 지수적으로 많음
- 사후분포의 형태가 복잡·비정형

## 2. Variational Inference (변분 추론)

### 2.1 아이디어
진짜 `p(z|x)` 대신 **다루기 쉬운 분포 `q(z)`** 로 근사:
```
q*(z) = argmin_{q ∈ Q}  D_KL( q(z) ∥ p(z|x) )
```

### 2.2 KL Divergence 정의
```
D_KL(q ∥ p) = E_q[ log(q/p) ] = ∫ q(z) log(q(z)/p(z|x)) dz  ≥ 0
```
- 두 분포 간의 비대칭 "거리"
- q = p 일 때만 0

## 3. ELBO (Evidence Lower Bound)

### 3.1 유도
```
log p(x) = ELBO + D_KL( q(z) ∥ p(z|x) )
         ≥ ELBO                                (KL ≥ 0 이므로)
```
log p(x) 는 (데이터 고정이면) 상수. ELBO 최대화 ≡ KL 최소화 ≡ 좋은 근사.

### 3.2 ELBO의 두 항 분해
```
ELBO = E_q[log p(x|z)]   −   D_KL( q(z) ∥ p(z) )
         └── Reconstruction ──┘   └── Regularization ──┘
```

### 3.3 호랑이/늑대/가축 비유
- **호랑이** : 어려운 항 (log p(x|z) 등) — 직접 계산 불가
- **늑대** : 신경망으로 근사 가능한 항
- **가축** : 쉬운 항 (양쪽 가우시안 KL 같은 closed form)
- 전략 : 어려움을 피하고 쉬움을 최대화

## 4. Autoencoder 기반 해석

### 4.1 Deterministic Autoencoder
```
x → Encoder → z → Decoder → x̂
```
- 점-대-점 매핑 (결정적)
- Reconstruction Loss 최소화

### 4.2 Stochastic (VAE 원형)
```
x → Encoder → (μ, σ) → z 샘플 → Decoder → (μ_x, σ_x) → x̂ 샘플
```
- 점-대-분포 매핑 (확률적)
- 잠재 공간에 연속성과 완전성 부여

## 5. Mean Field Approximation

```
q(z₁, z₂, …, z_d) = ∏ᵢ qᵢ(zᵢ)
```
- 차원 간 독립 가정 (비현실적이나 계산 가능)
- Variational Inference의 고전적 전략

## 6. Parametrization (매개변수화)

### 6.1 신경망으로 분포 표현
- `q_φ(z|x)` : Encoder 신경망 (파라미터 φ)
- `p_θ(x|z)` : Decoder 신경망 (파라미터 θ)

### 6.2 최적화
φ, θ 를 **Gradient Descent** 로 **공동 학습** → ELBO 최대화.

## 7. 최종 Loss 형태

```
L_VAE = L_recon + L_KL
      = - E_q[log p_θ(x|z)]  +  D_KL( q_φ(z|x) ∥ p(z) )
```

- `L_recon` : 데이터 타입에 따라 MSE(연속) / BCE(이진) / CE(다중)
- `L_KL` : 두 가우시안 사이 closed form

## 8. MCMC vs Variational Inference

| 방법 | 특성 | Trade-off |
|------|------|-----------|
| **MCMC** | 샘플 기반, 점근적으로 정확 | 느림 |
| **Variational** | 최적화 기반, 근사적 | 빠름 |

- MCMC : 시각장애인이 맹목 탐색 후 코끼리 발견 → 주변 더듬기
- VI : 가능한 모양 가정(예: 원형) → 최적화로 맞춤

## 9. 지구 인구 추정 비유

정확히 알기 어렵다면 **하한(Lower Bound)** 으로 접근:
```
World pop ≥ 한국(5천만) + 일본(1억) + 중국(12억) + …
```
ELBO 최대화 = log p(x) 에 대한 하한을 조이며 추정을 개선.

## 10. 핵심 정리

1. `p(z|x)` 직접 계산은 일반적으로 Intractable.
2. VI는 q(z)로 p(z|x)를 근사, KL 최소화.
3. KL ≥ 0 (Gibbs 부등식).
4. ELBO = Reconstruction − KL.
5. `log p(x) ≥ ELBO` 이므로 ELBO 최대화 = log p(x) 간접 최대화.
6. Reconstruction 항 = 데이터 재구성 가능성.
7. KL 항 = prior 근접성 (정규화).
8. Mean Field : 차원 독립 가정.
9. Parametrization으로 φ, θ 신경망 공동 학습.
10. L_recon 은 데이터 타입별 상이 (MSE/BCE/CE).
11. L_KL 은 가우시안 쌍 closed form.
12. MCMC는 정확하되 느리고, VI는 빠르되 근사.
13. 호랑이/늑대/가축 = 어려움·근사·쉬움 항의 구분.
14. Stochastic Decoder가 생성력이 높음.
15. 이 모든 개념이 VAE로 이어지고, 다음 강의(260420)에서 완전판으로 다뤄짐.

### 강의 중 주요 직관·일화
- **무한 원숭이 정리 (재언급)** : 유한 자원 → Approximation 필연.
- **지구 인구 추정** : 하한으로 근사하는 ELBO 직관.
- **추리소설 구조** : 작가는 단서(쉬운 항)를 강조하되 사건의 본질(어려운 항)을 간파.
- **호랑이·늑대·가축** : VI의 세 항을 난이도별로 분류하는 대표 비유.
- **시각장애인 코끼리** : MCMC와 VI의 철학 차이.
