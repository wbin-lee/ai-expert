### 2. 핵심 제안

사내 Claude Code **사용 로그(익명)** 를 분석해 우수 활용 사례를 검색·추천하고, 사용자 질문에 맞춤 코칭을 제공하는 시스템을 구축합니다.

1. **Ingest** — 로그 익명 정제, 세션·태스크 단위 정리, **토큰·비용(월 누적)** 메타 포함, 우수 세션 라벨링  
2. **PatternEmbed** — 우수·**저토큰 고효율** 패턴 임베딩 학습 (Contrastive FT)  
3. **CoachModel** — 활용·**예산 효율** 코칭·유사 사례 생성 (LoRA SFT)  
4. **Serve** — DS LLM API·게이트웨이 연동 (MCP: `suggest_usage`, `find_pattern`)

기존 활용 대시보드(`c4e.whchoi.net/analyze`)와 AWS 게이트웨이를 확장하며, **코칭 레이어**를 추가합니다.

**예산 효율 코칭 예시**
- 「레거시 분석인데 이번 달 예산 70% 썼어요」 → 저토큰 우수 사례·컨텍스트 축소·작업 분할 순서 제안  
- 「비슷한 태스크에서 토큰을 줄인 우수 사용자」 패턴 검색 및 안내
