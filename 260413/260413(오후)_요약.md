## 1. 역전파(Backpropagation) 의 핵심

### 1.1 목적과 원리
- 목적 : 손실 L 을 각 파라미터로 미분 `∂L/∂wᵢ`
- 원리 : **Chain Rule** 기반 역방향 미분
- 효율 : Forward 1회 + Backward 1회 — `O(파라미터 수)`

### 1.2 Chain Rule
```
dy/dx = (dy/dz) · (dz/dx)
```
신경망은 여러 함수의 합성 → Chain Rule 이 반복 적용된다. Computational Graph 로 시각화 가능.

### 1.3 Cache 메커니즘
Forward 에서 계산한 중간값을 Backward 에서 **재사용** (재계산 피함).
- 곱셈 : `c = a · b` → `∂L/∂a = b · ∂L/∂c` 이므로 b를 캐시
- Sigmoid : σ(z) 를 캐시하면 σ'(z) = σ(1-σ) 로 빠르게 구함

## 2. Gate Operation의 Backward

### 2.1 덧셈 gate
```
Forward  : c = a + b
Backward : ∂L/∂a = ∂L/∂c,  ∂L/∂b = ∂L/∂c    (gradient 분산)
```

### 2.2 곱셈 gate
```
Forward  : c = a · b
Backward : ∂L/∂a = b · ∂L/∂c,  ∂L/∂b = a · ∂L/∂c   (cache 필요)
```

### 2.3 Sigmoid gate
```
Forward  : y = σ(z)
Backward : ∂L/∂z = σ(z)(1-σ(z)) · ∂L/∂y
```

## 3. 2-Layer MLP 실전 구현 (Softmax + Cross-Entropy)

### 3.1 Forward
```
z1 = W1 · x + b1
a1 = ReLU(z1)
z2 = W2 · a1 + b2
ŷ  = softmax(z2)
L  = -Σ y · log(ŷ)       (Cross-Entropy)
```

### 3.2 Backward (역순)
```
dz2 = ŷ − y                           # softmax + CE 간단 형태
dW2 = dz2 · a1ᵀ
db2 = Σ dz2                           # 배치 합

da1 = W2ᵀ · dz2
dz1 = da1 ⊙ (z1 > 0)                  # ReLU 미분 = 스위치
dW1 = dz1 · xᵀ
db1 = Σ dz1
```

## 4. 가중치 초기화

### 4.1 왜 중요한가
- 나쁜 초기화 → 신호가 0 으로 수렴 또는 폭발
- 깊은 네트워크일수록 결정적

### 4.2 Xavier / Glorot
```
w ~ U[ -√(6/(n_in + n_out)),  +√(6/(n_in + n_out)) ]
```
- 목적 : 레이어 통과 시 분산 유지
- 적합 : Sigmoid, Tanh

### 4.3 Kaiming / He
```
w ~ N( 0,  2/n_in )
```
- 목적 : ReLU 의 절반 죽은 뉴런 보정 (변수 2배)
- 적합 : ReLU, Leaky ReLU

## 5. Batch Normalization

### 5.1 공식
```
x̂ = (x − μ_B) / √(σ²_B + ε)
y  = γ · x̂ + β              (학습 가능한 scale / shift)
```
- 배치 단위로 각 차원을 정규화
- γ, β 는 학습 파라미터

### 5.2 효과
1. **그래디언트 흐름 개선** → 깊은 네트워크 학습 용이
2. **Learning Rate ↑** 가능
3. **정규화 효과** → Dropout을 일부 대체 가능
4. **초기화 민감도 감소**

## 6. Vanishing Gradient 문제

### 6.1 원인
Chain Rule 상 0~1 사이 미분값이 반복 곱해지면 0 으로 수렴:
```
∂L/∂w_early = ∏ (각 층의 미분 ∈ (0, 1)) → 0
```

### 6.2 해결책
- **ReLU** 활성화
- **Xavier / Kaiming 초기화**
- **Batch Normalization**
- **Residual Connection** (ResNet)

## 7. 학습 루프 표준

```python
for epoch in range(num_epochs):
    for x_batch, y_batch in train_loader:
        y_pred = model(x_batch)
        loss   = cross_entropy(y_pred, y_batch)

        loss.backward()
        optimizer.step()
        optimizer.zero_grad()

        # 주기적 Validation
        val_loss = evaluate(model, val_loader)
        if val_loss.improved: save_model()
        elif not_improving_too_long: early_stop()
```

## 8. 핵심 정리

1. Backprop = Chain Rule 기반 효율적 미분.
2. Forward 캐싱 없이는 Backward 불가.
3. 덧셈 gate = 분산, 곱셈 gate = 교차, Sigmoid gate = 스케일.
4. Softmax + Cross-Entropy → `dz = ŷ − y` (매우 간단).
5. ReLU Backward = 0/1 스위치.
6. 배치 차원은 항상 합산.
7. Xavier ↔ Sigmoid/Tanh, Kaiming ↔ ReLU.
8. BatchNorm = 정규화 + LR ↑ + 정규화 효과.
9. Vanishing Gradient는 깊은 네트워크의 주요 병목.
10. 해결 : ReLU + 적절한 초기화 + BN + Residual.
11. LR Warm-up은 대배치에서 필수.
12. 2층 MLP만으로도 많은 문제 해결 가능 (Universal Approximation).
13. 깊은 학습 = 초기화·활성화·BN 삼박자.
14. Cache는 메모리 vs 속도의 trade-off.
15. 백프로파게이션 구현은 프레임워크가 자동화 (PyTorch autograd).

### 강의 중 주요 직관·일화
- **요리사의 재료 보관 (Cache)** : Forward 에서 사용한 재료를 버리지 말고 Backward 에 재사용.
- **Vanishing = 위로 갈수록 줄어드는 시냇물** : BN과 ReLU가 "댐" 역할.
- **초기화는 경사 각도** : 너무 가파르면 눈사태, 너무 완만하면 멈춤.
