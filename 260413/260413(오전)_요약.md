# 260413 오전 강의 요약 — 신경원 모델·MLP·활성화 함수

## 1. 신경원(Neuron) 모델

### 1.1 3가지 핵심 요소
1. **가중합 (Weighted Sum)** : `z = Σᵢ wᵢ xᵢ + b = wᵀx + b`
2. **Bias** : 상수항 b (뉴런 발화 임계값 역할)
3. **활성화 함수** : 비선형성 부여

### 1.2 신경과학 용어와 공학 용어
- 신경과학 : **Synaptic Weight (시냅스 가중치)**
- 공학 : **Weight**
- 활성도(Activation) : 뉴런의 활성화 정도 (보통 0~1)

## 2. Logistic Sigmoid

### 2.1 공식
```
σ(z) = 1 / (1 + e^{-z})
σ'(z) = σ(z) · (1 - σ(z))
```

### 2.2 특성
- 범위 : (0, 1) → 확률 모델링 적합
- z ≤ -4.6 → 거의 0, z ≥ +4.6 → 거의 1로 포화
- 통계학 기원 : **Log-Odds = log(P(y=1)/P(y=0)) = wᵀx + b** 를 선형으로 모델링

## 3. 다층 신경망 (MLP) 아키텍처

### 3.1 구조
```
Input Layer → Hidden Layer(s) → Output Layer
(fully connected + feedforward, 피드백 없음)
```

### 3.2 특징
- **Fully Connected** : 모든 뉴런이 다음 층의 모든 뉴런과 연결
- **Feedforward** : 피드백 루프 없음, 방향 순방향만
- **Bipartite Graph** : 같은 레이어 내부엔 연결 없음

### 3.3 행렬 표기
- 가중치 행렬 W : (d × m), d=입력, m=출력
- 편향 벡터 b : (m,)
```
y = Wᵀx + b
또는 계층적 표기 :
z^(l) = W^(l) · a^(l-1) + b^(l)
a^(l) = φ( z^(l) )
```

### 3.4 파라미터 vs 하이퍼파라미터
- **파라미터** : W, b (학습 대상)
- **하이퍼파라미터** : 레이어 수, 뉴런 수, 활성화 함수 (사람이 설정)

## 4. 활성화 함수 진화

| 함수 | 공식 | 특징 |
|------|------|------|
| Step | 0 or 1 | Perceptron 초기 |
| Logistic Sigmoid | σ(z) = 1/(1+e⁻ᶻ) | 0~1, saturation 문제 |
| Tanh | (eᶻ-e⁻ᶻ)/(eᶻ+e⁻ᶻ) | -1~1, 더 나음 |
| **ReLU** | max(0, z) | 현대 표준 |
| Leaky ReLU | z if z>0 else αz | 죽은 뉴런 문제 완화 |
| GELU | Gaussian 근사의 매끈 버전 | Transformer 계열 |
| SwiGLU | Swish + Gated Linear Unit | 현대 LLM에서 사용 |

## 5. 합성함수(Composite Function)로서의 MLP

```
f(x; θ) = f₃( f₂( f₁(x) ) )
```
- 고등학교 합성함수와 동일 원리
- 깊이가 표현력을 결정

## 6. Universal Approximation Theorem

**충분히 많은 히든 뉴런 + 비선형 활성화가 있으면, 임의의 연속 함수를 원하는 정확도로 근사 가능.**
- MLP의 이론적 근거
- "데이터와 계산만 충분하면 임의의 함수 표현 가능"을 보장
- 단, 실제 학습은 데이터·최적화 별개 문제

## 7. Logistic Regression vs MLP

- **Logistic Regression** = 출력층만 있는 1층 MLP (0개 히든)
- **MLP** = 히든층(들) 추가로 비선형 결정 경계 형성

## 8. 핵심 정리

1. 뉴런 = 가중합 + 편향 + 활성화 함수.
2. Sigmoid는 확률·Log-Odds 모델링.
3. ReLU가 현대 표준 (계산 효율, 미분 안정).
4. MLP = Fully Connected + Feedforward.
5. W:(d×m), b:(m,) 행렬 표기.
6. Bipartite Graph 구조 (같은 레이어 내부 미연결).
7. 파라미터(w,b)는 학습, 하이퍼파라미터는 사람 설정.
8. 합성함수 구조가 깊이의 본질.
9. Universal Approximation (1989) 이 이론 근거.
10. 그러나 학습은 데이터·최적화·정규화 별개.
11. Logistic Regression = 1층 MLP (히든 0).
12. 각 레이어는 서로 다른 수준의 추상화 학습.
13. MLP는 CNN·RNN·Transformer 등 모든 현대 구조의 기본 블록.
14. 활성화의 비선형성이 층을 쌓는 의미를 만든다.
15. Bias는 뉴런의 발화 임계값 조절 장치.
