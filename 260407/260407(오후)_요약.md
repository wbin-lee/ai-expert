# 260407 오후 강의 요약 — ANN 역사·뇌 vs 컴퓨터·Backpropagation 심화

## 1. 인공신경망의 역사

### 1.1 주요 타임라인
| 연도 | 사건 |
|------|------|
| 1943 | McCulloch-Pitts : 뉴런 = AND/OR/NOT 논리 게이트 증명 |
| 1958 | Rosenblatt Perceptron (하드웨어 구현) |
| 1969 | Minsky & Papert : XOR 한계 증명 → 1차 AI 겨울 |
| 1982 | Hopfield Network : 연상 기억·에너지 기반 |
| 1985 | Boltzmann Machine / 이후 RBM |
| 1986 | Backpropagation 재발견 (Rumelhart·Hinton·Williams) |
| 1989 | Universal Approximation Theorem |
| 2006 | Hinton Deep Belief Network → 딥러닝 시작 |
| 2012 | AlexNet ImageNet 우승 |
| 2016~ | AlphaGo, Transformer, ChatGPT |

### 1.2 AI 패러다임 재조명
1. **Symbolism (1950~80)** — 규칙 기반·논리
2. **Connectionism (1980~현재)** — 신경망·데이터
3. **Embodied / Physical AI (현재~)** — 환경 상호작용, 휴머노이드

## 2. 뇌 vs 디지털 컴퓨터

### 2.1 구조·기능 비교
| 특성 | 뇌 | 디지털 컴퓨터 |
|------|-----|----------------|
| 단위 | 10¹¹ 뉴런 (느림, ms) | CPU/GPU (빠름, ns) |
| 메모리 | 분산·robust | 주소 기반·fragile |
| 주소 | Content-addressable | Location-addressable |
| 신호 | 아날로그 | 디지털 |
| 연산 | 내적(유사도) | 논리·산술 |
| 에너지 | 25 W | 수백 W~kW |

### 2.2 뇌의 4가지 핵심 특성
1. **Parallel Distributed Processing** : 10¹⁴ 시냅스, 대규모 병렬
2. **Selective Attention** : 무한 정보 중 중요한 것만 선택
3. **Adaptive Plasticity** : 신경 가소성으로 재구성
4. **Memory-Centric** : 처리보다 메모리가 근본

### 2.3 Neuromorphic Computing
뇌의 병렬성·에너지 효율을 모방하는 차세대 하드웨어 방향.

## 3. Hopfield Network와 Boltzmann Machine

### 3.1 Hopfield Network (1982)
- 완전연결 신경망 (대칭 가중치)
- **Lyapunov 에너지 함수** 감소 → 안정 수렴 보장
- **연상 기억 (Associative Memory)** 구현
- 기억 용량 : 뉴런 N개일 때 약 **0.15N** 패턴

### 3.2 Boltzmann Machine & RBM
- Hopfield 확장 : **히든 뉴런 + 확률적 동작**
- RBM (Restricted) : 층 간만 연결 → 학습 가능
- 이후 Deep Belief Network(DBN, 2006)로 발전

## 4. Backpropagation 수식

### 4.1 손실함수 (MSE 예시)
```
E = ½ Σ_d (y_d - f_d)²
```

### 4.2 가중치 업데이트
```
∂E/∂wᵢ = -(y_d - f_d) · xᵢ = -error · input
Δwᵢ = -η · ∂E/∂wᵢ
wᵢ(new) = wᵢ(old) + Δwᵢ
```

### 4.3 Chain Rule (다층)
```
∂E/∂w = (∂E/∂f) · (∂f/∂s) · (∂s/∂w)
```
- Forward Pass : 입력 → 예측
- Backward Pass : Loss → 역방향으로 그래디언트 누적

## 5. 층(Layer) 깊이와 표현 능력

| 구조 | 가능한 결정 경계 |
|------|------------------|
| 단층 (Perceptron) | 직선 (선형) |
| 2층 (1 히든) | 곡선 (비선형) |
| 다층 (Deep) | 복잡한 함수, 추상적 특징 계층 |

단, 깊을수록 **Vanishing Gradient** 문제 심화 → 이후 ReLU·BatchNorm·Residual로 해결.

## 6. 활성화 함수 (상세)

| 함수 | 식 | 특징 |
|------|-----|------|
| Sigmoid | σ(x) = 1/(1+e⁻ˣ) | 0~1, 미분 0~0.25 → vanishing |
| Tanh | (eˣ-e⁻ˣ)/(eˣ+e⁻ˣ) | -1~1, Sigmoid보다 나음 |
| **ReLU** | max(0, x) | 현대 표준, 미분 상수 1 (z>0) |
| Softmax | eˣᵢ/Σeˣⱼ | 다중 클래스 확률 출력 |

## 7. Universal Approximation Theorem (1989)
**충분한 히든 뉴런이 있고 비선형 활성화가 있으면, 임의의 연속 함수를 임의의 정확도로 근사 가능.**
→ MLP의 이론적 근거, "데이터만 많으면 충분"의 수학적 보장.

## 8. Learning Rate의 역할
```
w_new = w_old - η · ∂E/∂w
```
- η 너무 큼 : 발산, 최적점 건너뜀
- η 너무 작음 : 학습 느림, 로컬 최솟값
- η 적절 : 빠르고 안정적 수렴

## 9. 핵심 정리

1. 신경망은 뇌의 망 구조 모방, 여전히 디지털로 구현.
2. Hopfield = 연상 기억, 용량 ≈ 0.15N.
3. RBM/DBN = 확률적 신경망 + layer-wise 사전학습.
4. Backprop = Chain Rule 기반 역전파, O(parameter).
5. 단층 vs 다층 = 직선 vs 곡선 경계.
6. ReLU 등장으로 Vanishing Gradient 완화.
7. Universal Approximation (1989) = MLP 이론 근거.
8. Learning Rate 튜닝이 수렴의 핵심.
9. Memory-centric : CPU보다 HBM·메모리가 병목.
10. Gradient Descent → Local Optimum (Global 보장 X).
11. 3번의 AI 겨울을 거쳤지만 패러다임 시프트로 부활.
12. 현재 Connectionism 승리, 다음은 Physical AI.
13. End-to-End 학습 = 중간 처리 자동 설계.
14. NIPS 참가자 폭증(200→수만명)이 AI 부흥의 상징.
15. Hinton·Hopfield의 2024 노벨 물리학상이 신경망을 주류 과학으로 편입.

### 강의 중 주요 직관·일화
- **Minsky의 역설** : 신경망은 불가능하다 주장했으나 50년 후 신경망 승리. 과학도 정치적이라는 교훈.
- **Rumelhart의 비극** : Backpropagation 공동 개발자. 신경 질환으로 사망 → Rumelhart Prize 제정.
- **Deep Blue vs AlphaGo** : Deep Blue(1997)는 탐색·하드웨어, AlphaGo(2016)는 신경망. 20년 = 패러다임 차이.
- **Google 자율주행차** : 2011 발표 이후 15년 지나도 미완성. "마지막 1%(Physical AI)가 가장 어렵다."
- **와인 전문가가 된 ChatGPT** : 텍스트로만 학습 → 실제 미각 grounding 부재. Hallucination 발생의 한 원인.
