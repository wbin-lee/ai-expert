# 260421 오전 강의 요약 - GAN (Generative Adversarial Network) 완전판

## 1. 복습 : 지금까지 배운 생성형 모델 지도

### 1.1 큰 그림
뽑기 좋은 분포(가우시안 등)에서 **먼지 z** 를 하나 뽑고, **디코더(또는 제너레이터)** 에 통과시키면 "손오공"(의미 있는 x)이 나오게 하는 구조는 공통.

### 1.2 모델 비교
| 모델 | z 만드는 방법 | 목적함수 | 이론적 배경 |
|------|---------------|----------|-------------|
| **Autoregressive** | chain rule로 P(x₁)·P(x₂\|x₁)·… 분해 | Next-token likelihood | Likelihood-based |
| **VAE** | Encoder가 q_φ(z\|x) 로 z 생성 | ELBO (lower bound) | **Variational Inference** |
| **GAN** | 디코더=Generator + Discriminator가 진위 판단 | DGZ 기반 minimax | **Game Theory** |
| **Diffusion** | 노이즈 추가/역과정 학습 | Noise prediction | Score / SDE |

- VAE의 발전 : 적분을 Variational Inference로, 인코더가 "똘똘한 z"를 고르는 역할.
- GAN의 발전 : P(x)를 직접 계산하기 어렵다면 **샘플링 능력만 학습** 하면 되지 않겠는가? 라는 발상.

## 2. GAN의 철학 : Game Theory

### 2.1 게임 이론의 창시자 — John Nash
- 1950~60년대 프린스턴 박사과정 시절 **게임 이론(Game Theory)** 을 만들고 MIT 교수가 된 수학자.
- Schizophrenia(조현병) 진단 후 정년 보장 직전 학교에서 쫓겨나는 극적 비극 → 훗날 이론이 경제학 주류로 자리잡아 **노벨 경제학상** 수상.
- 영화 **"Beautiful Mind"** 로 일대기가 그려짐.

### 2.2 시장 경제와 게임 이론
- 시장에는 **파는 사람(Seller)** 과 **사는 사람(Buyer)** 두 플레이어가 존재.
- 파는 사람은 가격을 **maximize**, 사는 사람은 **minimize** → 타협점(**Nash Equilibrium**)에서 거래 성립.
- GAN의 핵심 = **Minimax Game** : 한쪽은 min, 한쪽은 max.

### 2.3 가위바위보의 교훈
- 부모가 "3분의 1 확률로 내라"고 가르친 적 없지만, 아이들은 **러닝**을 통해 골고루 내야 함을 체득.
- Nash 균형(각 선택 1/3)에 자연스럽게 수렴하는 과정이 **경험적 학습(empirical learning)** 과 정합.

## 3. GAN의 두 주인공

### 3.1 Generator (G)
- 뉴럴넷. 파라미터 **θ_G**.
- 입력 : 먼지 z (가우시안 / 유니폼 분포에서 샘플).
- 출력 : **G(z)** = 가짜 데이터 x̃.
- 역할 : 진짜처럼 보이는 샘플을 만들어 판별자를 속이는 것 (파는 사람).

### 3.2 Discriminator (D)
- 뉴럴넷. 파라미터 **θ_D**.
- 입력 : 실제 x 또는 가짜 G(z). **차원은 반드시 같음** (가짜 돈도 진짜와 크기가 같아야 위패 감별기에 들어간다).
- 출력 : **진짜일 확률** (0 ≤ D(·) ≤ 1).
- 역할 : 진짜와 가짜를 구별하는 감별자(사는 사람).

### 3.3 Ian Goodfellow — GAN의 창시자
- 스탠포드 학부(CS) + 아마도 Economics 부전공 → 경제학에서 게임 이론을 접했을 가능성.
- 대학원은 명문 대신 **몬트리올 대학** 요슈아 벤지오 지도 하에 진학 → 2014년 GAN 논문.
- Bengio 연구실의 파격적 문화 : 학생 70+ 명 대비 책상 40개 (**first-come first-served**).

## 4. DGZ — GAN의 "가격"

### 4.1 핵심 네 가지 확률
| 기호 | 의미 |
|------|------|
| **D(x)** | 진짜를 **진짜**라고 하는 확률 |
| **1 − D(x)** | 진짜를 **가짜**라고 하는 확률 |
| **D(G(z))** | 가짜를 **진짜**라고 하는 확률 ← **DGZ (가격)** |
| **1 − D(G(z))** | 가짜를 **가짜**라고 하는 확률 (발각 확률) |

### 4.2 "DGZ = 가격" 비유
- 시장경제에서 가격 흥정이 핵심인 것처럼, GAN의 흥정 대상은 **D(G(z))**.
- Discriminator(사는 사람) : **DGZ를 최소화** (싸게 사려 한다).
- Generator(파는 사람) : **DGZ를 최대화** (비싸게 팔려 한다).

## 5. Nash Equilibrium : GAN의 수렴 지점

- **G의 균형** : 만들어낸 분포 = 실제 데이터 분포. 즉, 가짜와 진짜를 구별 불가.
- **D의 균형** : 모든 입력에 대해 **D(·) = 0.5** (동전 던지기).
- 주의 : "D가 완벽히 구별할 때"가 아니라, 오히려 G가 너무 잘 만들어서 **D가 역할을 못할 때**가 균형.

## 6. Training Objective

### 6.1 Discriminator의 목적
명확 : **진짜는 진짜라고, 가짜는 가짜라고** 한다.
```
J_D = E_{x~p_data}[ log D(x) ] + E_{z~p(z)}[ log (1 − D(G(z))) ]
```
- D는 이 항을 **maximize** (진짜·가짜 분류 정확도 ↑).
- Self-supervised 레이블링 : 진짜 데이터 → 1, 가짜 → 0. 그 후 **Cross-Entropy Loss** 로 학습.

### 6.2 Generator의 두 가지 선택지
- **Game-theoretic (원래 논문 앞부분) : min-max**
  ```
  min_G  E[ log (1 − D(G(z))) ]           # 발각 확률 minimize
  ```
- **Intuitive / Non-saturating (논문 뒷부분의 "고해성사") : max-max**
  ```
  max_G  E[ log D(G(z)) ]                 # 속일 확률 maximize
  ```

### 6.3 Minimax Objective (원 논문 표기)
```
min_G  max_D  V(D, G) = E_{x}[ log D(x) ] + E_{z}[ log (1 − D(G(z))) ]
```
- 빨간색(D의 맥시마이즈) + 파란색(G의 미니마이즈) 동시 진행.
- **이거 하나만 기억해도 GAN의 핵심.**

## 7. "고해성사" — Non-saturating Loss

### 7.1 왜 미니맥스는 잘 안 될까
- `log(1 − D(G(z)))` 를 D(G(z))의 함수로 그려보면 :
  - **D(G(z)) ≈ 0** (G가 아직 엉망) 구간 → **기울기가 거의 0** → 학습 신호 미약.
  - **D(G(z)) ≈ 1** (G가 잘함) 구간 → 기울기가 매우 큼 → 이미 잘하는 애를 스파르타식으로 굴림.
- 비유 : **"은퇴자 반은 딥러닝 스파르타식, 초등학생 반은 6년 팽이 놀아"** 같은 뒤집힌 교육.

### 7.2 해결책 : 뒤집기
- `1 − D(G(z))`를 minimize 하는 대신, `D(G(z))`를 **maximize**.
- **수학적으론 조삼모사지만, 그래디언트 측면에서 학습이 훨씬 안정**.
- Goodfellow는 논문 앞에서는 game-theoretic으로 팔고(Hollywood show), 뒤쪽 실험 파인프린트로 **"실제로는 max-max가 낫다"** 이실직고 → "이실직고 했기에 존경".

## 8. Training Algorithm (원 논문)

### 8.1 Alternating Update
```
for number_of_iterations :
    # --- Discriminator 업데이트 (K번 반복) ---
    for k in range(K) :
        z_1..m ~ p(z)        # 가짜 노이즈 m개
        x_1..m ~ p_data      # 진짜 데이터 m개
        θ_D ← θ_D + η · ∇_{θ_D} [ (1/m) Σ log D(x_i) + log(1 − D(G(z_i))) ]

    # --- Generator 업데이트 (1번) ---
    z_1..m ~ p(z)
    θ_G ← θ_G − η · ∇_{θ_G} [ (1/m) Σ log(1 − D(G(z_i))) ]      # 고해성사 전
    # (실제로는 θ_G ← θ_G + η · ∇_{θ_G} [ (1/m) Σ log D(G(z_i)) ] 고해성사 후)
```

- K는 hyperparameter (부모 마음 🙂).
- **D를 K번, G를 1번** : "큰애(D)는 운동장 K바퀴, 둘째(G)는 1바퀴" 비유. D가 약간 앞서가야 G에게 학습 신호를 줄 수 있음.

### 8.2 G 업데이트에서 D를 통과해야 하는 이유
- G의 loss `L_G = log D(G(z))` 는 **D가 있어야 계산됨**. 정답 레이블이 G에 직접 있지 않고 D의 출력을 통해 나옴.
- 그래서 백프롭은 **D를 통과** 하지만, **θ_D는 freeze** (이 시간은 G의 시간).
- 체인룰 전개 (슬라이드) :
  ```
  ∂L_G / ∂θ_G = (∂L_G / ∂D(G(z))) · (∂D(G(z)) / ∂G(z)) · (∂G(z) / ∂θ_G)
               └─────A──────┘   └──────B───────┘   └────C────┘
  ```
  A : 스칼라 스케일러, B : D의 로컬 그래디언트 (통과만), C : G의 그래디언트 (실제 업데이트).

## 9. GAN vs VAE 퀄리티 — 시대의 승자

### 9.1 2014년 ~ 2016년의 반전
- VAE가 먼저 주목받았지만, **GAN이 압도적 퀄리티** 로 곧 VAE를 추월.
- **StyleGAN** (Tero Karras, NVIDIA) : 거의 완벽한 FHD 인물 사진. 디퓨전과 지금도 대등한 성능.

### 9.2 GAN vs Diffusion (현재)
| 항목 | GAN | Diffusion |
|------|-----|-----------|
| 샘플 퀄리티 | 우수 | 우수 (약간 앞) |
| 다양성 (diversity) | 떨어짐 (mode collapse) | 더 좋음 |
| Controllability | 떨어짐 | **매우 우수** (conditional generation) |
| 샘플링 속도 | **빠름** (one-shot) | 느림 (iterative) |
| 학습 안정성 | **불안정** (minimax) | 매우 안정 |
| 학습 비용 | 중 | **높음** |

- **음성 합성** 등 **인퍼런스 속도**가 중요한 곳, **반도체 웨이퍼** 같이 대량 생성이 필요한 곳에서는 여전히 GAN이 유리.

### 9.3 역사적 이정표
- **2014** : Goodfellow GAN 논문.
- **2015** : Sohl-Dickstein이 DPM 발표 (성능 미약).
- **2018** : 크리스티 경매에서 최초 **GAN 그림이 $432,500**에 팔림. 중세 유화 학습 → 그림 하단 서명은 **GAN이 그리지 못해 사람이 쓴 것** (글씨 생성의 난점 엿보임).
- **DDPM** (Jonathan Ho, Berkeley) 이후 Diffusion이 재탄생하며 메인스트림으로 등극.
- **Score-based diffusion** : Yang Song (칭화대 물리 → Stanford CS).

## 10. Performance Evaluation — Quality/Diversity 측정

### 10.1 Quality-Diversity Trade-off (엔트로피 기반 해석)
- **엔트로피 ↑** = 다양성 ↑, 예측 어려움 ↓ (Uniform distribution이 최대).
- **엔트로피 ↓** = 예측 가능 (Dirac delta가 최소).

| 측면 | 원하는 분포 | 엔트로피 |
|------|-------------|----------|
| **Quality** (하나의 x에 대해 레이블을 잘 맞춰야) | p(y\|x) 가 **한 클래스에 쏠림** | 낮을수록 좋음 |
| **Diversity** (여러 x를 만들었을 때 다양한 클래스) | p(y) 가 **평탄** | 높을수록 좋음 |

### 10.2 Inception Score (IS)
```
IS = exp( E_x [ KL( p(y|x) ∥ p(y) ) ] )
```
- p(y|x) 뾰족할수록(quality ↑), p(y) 평평할수록(diversity ↑) 두 분포가 많이 다름 → KL ↑ → IS ↑.
- 높을수록 좋음. Exponential은 양수 비교를 쉽게.
- **왜 Inception?** 확률을 구하려면 분류기가 필요했는데 그 당시 대세가 **Inception (GoogLeNet)**.
- **단점** :
  1. KL의 비대칭성 → 반쪽짜리 측정.
  2. 클래스 당 한 종류만(서브클래스 무시) 내놔도 높게 나옴 — "차 그리라 했는데 계속 제네시스만 그려도 OK".

### 10.3 Frechet Inception Distance (FID)
- 선형대수에서 두 Gaussian 분포 사이의 거리로 이미 존재하던 개념.
- 이미지 → Inception v3의 `fc3` 층 (2048차원) activation의 분포를 Gaussian으로 근사.
- 진짜 / 가짜 각각 μ, Σ 를 구하고 :
  ```
  FID = ‖μ_r − μ_g‖² + Tr(Σ_r + Σ_g − 2(Σ_r Σ_g)^{1/2})
  ```
- **낮을수록 좋음**. 보통 **IS와 FID를 함께 리포트** (2D 비교).

### 10.4 Precision / Recall / F1
- **Precision** = 내가 만든 것 중 진짜 같은 것의 비율 → **Quality**.
- **Recall** = 전체 데이터 중 얼마나 재현했나 → **Diversity**.
- **F1** = 둘의 harmonic mean.
- 이미지에서는 상당 부분 **정성적(qualitative)** 평가.

### 10.5 MOS (Mean Opinion Score)
- 음성 합성 등에서 사람이 직접 평가. Amazon Mechanical Turk(AMT) 등 활용.

## 11. Mode Collapse — GAN의 고질병

### 11.1 현상
- 다양한 z를 넣어도 **비슷한 출력**만 나오는 현상.
- 완전 동일은 드물고 대개 **Partial Mode Collapse** (시스터리처럼 느낌만 비슷).
- ImageNet 1000 클래스 학습 시 G가 10가지 정도만 만들다 뻗는 문제.

### 11.2 원인 가설 (여러 논문)
1. **데이터**의 다양성 부족.
2. **트레이닝 과정**의 불균형.
3. **max-min ≠ min-max** 문제.
4. **모델 capacity** 부족.
5. **Loss function** 이슈 (JSD → WGAN으로 가는 동기).

## 12. GAN 안정화의 걸작들

### 12.1 Wasserstein GAN (WGAN)
- **기원 일화** : Stanford 기계공학과 포닥 시절, 오스트리아인 지도교수가 화장실에서 **"Vat are you vorking on?"** 이라고 물었는데, 독일어 발음 때문에 **"Wasserstein"** 으로 들린 것이 계기 (교수 본인의 재미있는 일화).
- 핵심 : 거리 측도를 **JS-Divergence → Earth-Mover's Distance (Wasserstein Distance)** 로 교체.
  - JS-Divergence : KL 두 방향의 합. KL이 교환법칙 성립 안 하므로.
  - Wasserstein (Earth-Mover) : **흙무더기를 한 쪽에서 다른 쪽으로 최소 에너지로 옮기는 거리**. Optimal Transport 분야.
- **Discriminator → Critic** 으로 개명 : 확률이 아니라 **스코어**를 출력.
- **장점** :
  1. Loss landscape가 매끄러움 → 학습 안정.
  2. **Loss ↔ 퀄리티** 상관관계가 높아 **디버깅 용이** (겐의 기존 문제 : loss 떨어져도 음성 이상함).
  3. D가 이미 잘해도 계속 학습 가능.
- 파생 : **WGAN-GP (Gradient Penalty)** 등.

### 12.2 Spectral Normalization (SN-GAN)
- **일본 연구자들** (Miyato, Kataoka, Koyama, Yoshida) 의 기여.
- 배경 : 원조 GAN으로 ImageNet을 학습하려 하면 **수 주간 발산** 없이 돌아가는 것이 어려웠음.
- 방법 : 각 layer weight matrix `W`의 **Singular Value Decomposition** 후 **가장 큰 singular value(spectral norm)** 로 `W`를 나눠줌.
  ```
  W_SN = W / σ_max(W)
  ```
- Transformer의 `√d` scaling과 같은 스피릿 : 그래디언트가 너무 뾰족하지 않게.
- 효과 : **몇 주간 발산 없이 학습**되어 고품질 이미지 생성 가능 → 실탄이 장전된 총의 막힌 총구를 뚫어준 기법.

## 13. VAE와 GAN의 하이브리드

### 13.1 각자의 장단점
| 구분 | VAE | GAN |
|------|-----|-----|
| **장점** | x → z 찾기 쉬움 (Encoder 존재), 해석 용이 | **Sharp, high-quality** 샘플 |
| **단점** | 블러리 | Inverse GAN이라는 분야가 있을 만큼 z 찾기 어려움, Mode collapse |

- **Inverse GAN** : 주어진 이미지를 만들 수 있는 z를 거꾸로 찾는 분야. Interpolation, 얼굴 편집 등에 필요 → 픽셀 공간 편집은 어렵지만 latent 공간 편집은 쉽다.

### 13.2 VAE + GAN 하이브리드
- Encoder(VAE식) + Discriminator(GAN식) 결합.
- 수백 편의 파생 모델 존재 : **Adversarial VAE, VAE-GAN, BiGAN, ALPHAGAN** 등.
- 로스 :
  ```
  L_prior (encoder를 간단한 분포로) + L_likelihood (데이터 likelihood) + L_GAN
  ```
- 주의 : "Loss"라 쓰여도 **maximize** 하는 항일 수 있으니 부호 확인 필수.

## 14. GAN → SAGAN 5년간의 혁신 (2014 → 2018)

Goodfellow가 2018 CVPR 튜토리얼에서 직접 정리한 5가지 주요 개선:

1. **Depth + Convolution** : Simple 아키텍처에서 **DCGAN** 류로 복잡화.
2. **Conditional Generation** : `G(z, y), D(x, y)` 형태로 **클래스 조건**을 넣음 → 집중적으로 학습 가능.
3. **Spectral Normalization** : 위 12.2 참조.
4. **TTUR (Two Time-scale Update Rule)** : Schmidhuber 제자 Hochreiter 팀. **G와 D의 learning rate를 분리** → 학습 밸런스 개선.
5. **Self-Attention** : Transformer가 나온 시점. CNN generator만 쓰면 **말의 엉덩이가 두 개로 붙는** 글로벌 불일치 문제 → self-attention으로 **non-local** 연산 추가 → **SAGAN**.

### 14.1 Goodfellow CVPR 튜토리얼 에피소드
- Schmidhuber (long-standing controversial figure)가 튜토리얼 중 손을 들고 "**Do you know what, 1978년에 누가 비슷한 거 했는데?**" 라며 본인 과거 연구를 계속 주장.
- Goodfellow가 결국 "**사람들은 네 질문이 아니라 내 튜토리얼을 들으러 왔다**"며 제지. YouTube에 영상 존재.

## 15. 주요 변종 GAN

### 15.1 CycleGAN (2017)
- **Unpaired Image-to-Image Translation**.
- **Paired** (pix2pix) vs **Unpaired** (CycleGAN) :
  - Paired : 스케치 ↔ 실사처럼 **1:1 대응 쌍** 필요 (데이터 구성 어려움).
  - Unpaired : 두 도메인의 이미지 집합만 있으면 됨 (예: 여름 사진 ↔ 겨울 사진).
- 구조 : 4개의 뉴럴넷 (G_A→B, G_B→A, D_A, D_B).
- **Cycle Consistency Loss** : `x → G(x) → F(G(x)) ≈ x`.
  - 비유 : "우주/타임머신을 다녀와도 똑같은 사람이어야 한다".
- 데모 사례 : 얼룩말 ↔ 말, 사진 ↔ Monet 화풍, 여름 ↔ 겨울.

### 15.2 Progressive GAN (ProGAN / PGGAN, Tero Karras)
- 아이디어 : **Low-resolution → High-resolution 점진적 학습**. 얼굴 그림에서 구도 먼저 → 디테일 순.
- 4×4 → 8×8 → 16×16 → ... → **1024×1024** (최초의 FHD 가짜 사람 사진, 솜털·땀구멍까지).
- 학습 안정성 ↑, 학습 시간 ↑ (그래도 **40일** 소요 — NVIDIA 답게 GPU 많이 먹음).
- Diffusion의 **Cascading** 과 연결된 아이디어.

## 16. Neural Style Transfer & AdaIN

### 16.1 Neural Style Transfer (Gatys, 2016)
- CNN을 분석해보니 feature map이 **Content(클래스, 객체)** 뿐만 아니라 **Style(질감, 재질)** 까지 인코딩.
- 그리고 둘을 **분리(separation)** 할 수 있음!
- 예시 : "나무집" + "쇠로 된 차" → style(나무) + content(차) 조합 → **나무차**.
- 대표 응용 : 반 고흐 화풍 + 사진 → 고흐풍 그림 (신문 보도).
- 한계 : **Prohibitively slow** (최적화 기반).

### 16.2 AdaIN (Adaptive Instance Normalization)
- 후학들이 분석 : Content / Style이 사실 **채널별 평균 μ, 분산 σ**에 담겨있더라.
- 방법 :
  ```
  AdaIN(x, y) = σ(y) · ((x − μ(x)) / σ(x)) + μ(y)
  ```
  - Content `x`를 normalize한 뒤 Style `y`의 μ, σ로 rescale.
- **실시간 (몇 ms)** 스타일 전이 가능 → 기존 대비 비약적 향상.
- StyleGAN의 핵심 빌딩블록.

## 17. StyleGAN — 오늘의 종착점

### 17.1 구조
- Baseline은 Progressive GAN.
- **Mapping Network** (FC 8층) : 원래의 단순 Gaussian `z` 를 **복잡한 latent `w`** 로 변환.
  - 왜? : 실제 얼굴 데이터는 latent 공간이 (남/여 × 단/장발) 등의 구조를 갖는데, 가우시안 공간에 그대로 쑤셔넣으면 **워핑(warping)** 발생 → 이상한 샘플 섞임.
  - Mapping network로 latent space를 데이터에 맞는 모양으로 재성형 가능.
- **각 resolution 레벨에 AdaIN + noise injection 삽입** :
  - 낮은 해상도(4×4) → **굵직한 style** : 인종, 성별, 얼굴 각도.
  - 높은 해상도 → **미세한 style** : 주근깨, 머리카락 한 올.
  - 추가 노이즈 → stochastic 디테일.
- **Style Mixing** : 두 이미지 A, B의 서로 다른 레벨의 style을 조합 → A의 성별/나이 + B의 머리카락/색 같은 믹스 생성.

### 17.2 FFHQ 데이터셋
- StyleGAN은 **Flickr-Faces-HQ** 데이터셋을 공개 (기여).
- 기존 CelebA-HQ 대비 인종·연령·액세서리 **다양성** 우수.
- 논문의 **데이터 트랙 기여**는 요즘 학회들이 별도 트랙을 둘 만큼 중요 (NeurIPS D&B Track 등).

### 17.3 결과
- 사람이 봐도 **가짜라고 구별 불가능**한 인물 사진 생성.
- Controllability의 초기 시도 → 이후 VLM, Nano Banana 같은 현대 편집 모델로 이어짐.

## 18. ML 엔지니어링의 세대 변천

강의에서 암시된 엔지니어 역할의 시대적 변화:

| 세대 | 엔지니어링 대상 | 시대 |
|------|-----------------|------|
| 1세대 | **Feature Engineering** | Pre-Deep Learning |
| 2세대 | **Architecture Engineering** | CNN / RNN 전성기 |
| 3세대 | **Loss / Objective Engineering** | GAN / Diffusion |
| 4세대 | **Prompt Engineering** | LLM 시대 (현재) |
| 5세대 (예상) | **Harness / Scaffolding Engineering** | 아직 정착 중 |

GAN 시리즈가 **Loss Engineering 시대의 대표 주자** — CycleGAN의 Cycle Consistency Loss, WGAN의 Wasserstein Loss, StyleGAN의 다양한 auxiliary loss 등이 그 증거.

## 19. 핵심 정리 (한 페이지 요약)

1. **철학** : GAN은 **Game Theory** 에 기반한 **Minimax Two-player Game**. Seller(G) vs Buyer(D).
2. **DGZ (D(G(z)))** = GAN의 "가격". G는 max, D는 min. 이 한 가지만 기억해도 반은 이해한 것.
3. **4가지 확률** : D(x), 1−D(x), D(G(z)), 1−D(G(z)) — 각각 진/가, 진/가 × 진/가 매트릭스.
4. **Objective** : `min_G max_D E[log D(x)] + E[log(1 − D(G(z)))]`. 실전에서는 G를 `max log D(G(z))` **non-saturating loss** 로.
5. **Nash Equilibrium** : G가 완벽해서 D = 0.5로 수렴할 때.
6. **Training** : D를 K번, G를 1번 교대로 업데이트. G 업데이트 시 D 통과하지만 θ_D는 freeze.
7. **Quality/Diversity Trade-off** : p(y|x)는 뾰족할수록, p(y)는 평평할수록 좋음 → **IS / FID** 로 평가.
8. **한계** : Training instability, Mode collapse, Blurry/한정된 다양성.
9. **안정화 걸작** : **WGAN** (Wasserstein distance), **Spectral Normalization**, **TTUR**, **Self-Attention**.
10. **하이브리드와 진화** : VAE + GAN 하이브리드 (수백 편), CycleGAN(Unpaired), ProGAN(점진적), StyleGAN(Mapping + AdaIN + Noise).

### 강의 중 주요 직관 비유 모음
- **먼지 / 손오공** : z → G → x 기본 구도.
- **파는 사람 / 사는 사람 / 가격 흥정** : Generator / Discriminator / DGZ.
- **위패 감별기와 가짜 돈** : D의 역할. 가짜 돈은 반드시 진짜와 **같은 차원**이어야 한다.
- **John Nash / Beautiful Mind** : Game Theory와 Nash Equilibrium의 기원.
- **가위바위보의 1/3** : Nash 균형은 **경험적 러닝**으로 체득되는 것.
- **큰애 vs 둘째** : D를 K번, G를 1번. 큰애는 스파르타식, 둘째는 말 안 들어도 공부하는 척만 해도 OK.
- **은퇴자 스파르타식 vs 초등학생 놀기** : Non-saturating loss를 쓰지 않았을 때의 그래디언트 뒤집힘.
- **Hollywood Show 와 Fine Print** : Goodfellow가 논문 앞에서는 game-theoretic으로 팔고 뒤에서 고해성사한 구조.
- **스키조프레니아의 John Nash** : 이론과 현실의 극적 괴리.
- **말 그림의 엉덩이가 두 개** : Local receptive field만 가진 CNN generator의 글로벌 불일치 → Self-Attention으로 해결.
- **우주/타임머신 갔다와도 똑같은 사람** : CycleGAN의 Cycle Consistency.
- **약·생약의 성분 추출** : Content와 Style을 분리해야 합성·재조합이 가능하다는 Neural Style Transfer의 정신.
- **중학교 미술시간 얼굴 그리기** : 눈 → 기준선 → 코 · 입 → 머리 순의 **탑-다운** 방식 = Progressive GAN의 아이디어.
- **나무차** : Content(차) + Style(나무) → Neural Style Transfer의 출발.
- **Wasserstein ↔ Working on?** : 오스트리아 지도교수의 독일어 발음이 WGAN의 이름을 낳은 일화.
