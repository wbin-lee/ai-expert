# 260406 오전 강의 요약 — AI 개요·역사·철학적 배경

## 1. AI의 정의와 분류

### 1.1 AI란 무엇인가
**AI의 정의** : 사람처럼 생각하고 행동하는 기계를 만드는 일. 더 넓게는 "사람이 하면 지능이 필요한 일을 기계가 할 수 있도록 하는 연구 분야".

과거 학계에서는 "Computational Intelligence" 또는 "Rationality"라는 용어를 선호했고, 최근 50~60년간 저널 제목에 AI가 거의 등장하지 않다가 최근 다시 우후죽순 나타나는 중.

### 1.2 AI의 4가지 분류
```
                        Thinking(사고)        Acting(행동)
Humanly(인간처럼)    Thinking Humanly      Acting Humanly
                    (인지과학, ChatGPT)     (Turing Test, Physical AI)
Rationally(합리적)   Thinking Rationally    Acting Rationally
                    (논리·추론, 고전 AI)   (에이전트, 산업화 중)
```

### 1.3 Turing Test와 Total Turing Test
- **Turing Test** : 기계가 인간과 구별되지 않는 대화 능력을 보이면 지능을 인정. ChatGPT는 현재 통과 수준.
- **Total Turing Test** : 물리적 행동까지 포함. Physical AI 의 목표.
- 관점 : "외부에서 관찰 가능한 대화·행동을 기준으로 기계 지능을 판단".

### 1.4 중국어 방 논증(Searle)
존 설(John Searle)의 비판 : **기호 조작(형식적 규칙 처리)만으로는 의미 이해나 마음이 생기지 않는다.** 튜링 테스트의 행동주의적 관점에 대한 대표 반론이며, Strong AI / Weak AI 논쟁의 중심에 있다.

## 2. 인공지능의 역사

### 2.1 주요 타임라인
| 연도 | 사건 | 의의 |
|------|------|------|
| 1936 | Turing Machine 제안 | 수학적 계산 가능성의 모델 |
| 1943 | McCulloch-Pitts 논문 | 신경세포로 AND/OR/NOT 구현 가능 → 디지털 vs 신경망 두 갈래 |
| 1956 | Dartmouth Conference | "AI" 용어 탄생 |
| 1958 | Rosenblatt Perceptron | 실제 하드웨어 구현 |
| 1969 | Minsky & Papert의 XOR 비판 | 단층 퍼셉트론 한계 증명 → 1차 AI 겨울 |
| 1982 | Hopfield Network | 연상 기억·에너지 기반 신경망 부활 |
| 1985~ | Boltzmann Machine, RBM | 확률적 신경망 |
| 1986 | Backpropagation 재발견 | 다층 신경망 학습 가능 |
| 1997 | Deep Blue vs Kasparov | 탐색 + minimax + 하드웨어로 체스 정복 |
| 2006 | Hinton Deep Belief Network | 딥러닝 시대 개막 |
| 2012 | AlexNet ImageNet 우승 | "딥러닝 모멘트" |
| 2016 | AlphaGo vs 이세돌 | 신경망 + MCTS로 바둑 정복 |
| 2017 | Transformer | Attention 기반 새 패러다임 |
| 2022 | ChatGPT | 생성형 AI 대중화 |

### 2.2 패러다임 전환
- 기호주의(Symbolic) → 연결주의(Connectionism) → 에이전트/Physical AI
- 지식 기반 → 머신러닝 → 표현 학습(딥러닝)

## 3. 뇌 vs 컴퓨터

### 3.1 구조적 차이
| 항목 | 뇌 | 디지털 컴퓨터 |
|------|-----|----------------|
| 기본 단위 | 10¹¹ 뉴런 | CPU/GPU |
| 연결 | 10¹⁴ 시냅스 | 버스(bus) |
| 속도 | ms (느림) | ns (빠름) |
| 메모리 | 분산 표현 | 국소 주소 기반 |
| 주소 방식 | Content-addressable | Location-addressable |
| 신호 | 아날로그, 불정확 | 디지털, 정확 |
| 에너지 | 약 25 W | 수백 W~kW |
| 연산 | 내적(유사도) | 논리/산술 |

### 3.2 폰 노이만의 유언 격 저술
폰 노이만은 1957년 병상에서 *The Brain and the Computer* 를 썼고, 이미 그 시점에 "디지털 vs 아날로그(뇌)" 비교와 뇌를 닮은 컴퓨터 필요성을 주장했다. 현대의 Neuromorphic Computing 방향성은 그 연장선에 있다.

## 4. 세 가지 AI 패러다임

### 4.1 Symbolism (기호주의, 1950~1980)
- 철학 : Rationalism, Deductive
- 방법 : 규칙 기반, 전문가 시스템, 형식 논리
- 문제점 : 지식 입력 비용 ↑, 예외 처리 어려움

### 4.2 Connectionism (연결주의, 1980~현재)
- 철학 : Empiricism, Inductive
- 방법 : 데이터 학습, 분산 표현, 병렬 처리
- 현재의 주류 (딥러닝)

### 4.3 Embodied / Physical AI (현재~미래)
- 철학 : Constructivism, 실천주의
- 방법 : 물리 환경 상호작용을 통한 학습
- 목표 : 휴머노이드 로봇, Physical AI

## 5. 철학적 논쟁

- **Functionalism (기능주의)** : 어떤 물질로든 지능 구현 가능 (기능이 같으면 OK)
- **Physicalism (물리주의)** : 정신도 물질로 환원 가능
- **Computationalism (계산주의)** : 마음은 기호 조작
- **Strong AI / Weak AI / AGI**
- **Singularity** : 기계가 인간 지능을 넘어서는 시점

## 6. AI의 멀티모달 진화

```
초기 : 언어(Language)
 ↓
추가 : 시각(Vision)
 ↓
확장 : 행동(Action / Robotics)
 ↓
미래 : 오감 기반 진정한 멀티모달
```

## 7. 핵심 정리

1. AI 궁극 목표 = 사람처럼 지각·사고·행동하는 기계.
2. Turing Test는 외부 행동만 보고 지능을 판단한다(내부 메커니즘 무관).
3. 중국어 방 논증은 "기호 조작만으로는 의미 이해 불가"를 지적한다.
4. 기호주의 = 명시적 지식 프로그래밍 vs 연결주의 = 데이터 학습.
5. 1950년대 예상(10~20년) vs 실제(48년 걸려 체스, 60년 걸려 바둑) — 지금은 오히려 예상보다 빠르게 진행.
6. 뇌(분산·병렬·아날로그)와 컴퓨터(국소·순차·디지털) 미스매치가 AI 난제의 본질.
7. 패러다임 순환 : 기호주의 → 연결주의 → Physical AI.
8. 데이터·컴퓨팅·알고리즘 3각이 AI 부흥의 전제.
9. "moving goalpost" : 풀린 문제는 더 이상 AI로 여겨지지 않는 특성.
10. Multimodal (Language·Vision·Action) + 오감이 다음 단계.
11. AI 세대론 : 1세대(규칙) → 2세대(ML+특징공학) → 3세대(딥러닝=표현 학습).
12. 1956 Dartmouth, 1986 Backprop, 2012 AlexNet, 2022 ChatGPT가 4대 변곡점.
13. 철학적 토대 : 기능주의·물리주의·계산주의.
14. 노벨상(Hopfield·Hinton)이 신경망을 물리학·과학의 주류로 편입시켰다.
15. AI 겨울은 기대-현실 갭에서 반복 발생했으며 각 회복은 패러다임 시프트였다.

### 강의 중 주요 직관 비유
- **"50년 과거 60년을 능가"** : 초기 연구자들의 과도한 낙관과 실제 성취 시점 비교.
- **폰 노이만의 병상 저술** : 1957년 이미 디지털 vs 아날로그를 비교한 선견.
- **Minsky의 아이러니** : 박사시절 신경망 연구자였으나 이후 신경망 비판을 이끈 장본인, 결국 연결주의 승리.
- **Physical AI의 마지막 1%** : 자율주행·로봇 등 "인지 + 행동 통합"의 어려움을 강조.
