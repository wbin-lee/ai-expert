# 260408 오전 강의 요약 — CNN·RNN 기초 및 실습

## 1. CNN (Convolutional Neural Network)

### 1.1 핵심 개념
**CNN** : 이미지 처리에 특화된 신경망. 인간이 사진을 볼 때 **지역적(local)** 정보를 모아 전체를 파악하는 방식을 모방한다. 로컬 정보를 필터로 추출 → 특징 맵(Feature Map) → 최종 분류.

### 1.2 주요 구성 요소
- **필터(Kernel)** : 로컬 정보를 수집하는 작은 가중치 매트릭스 (예 3×3, 5×5)
- **스트라이드(Stride)** : 커널의 슬라이딩 간격
- **패딩(Padding)** : 이미지 가장자리를 0으로 채워 정보 손실 방지
- **풀링(Pooling)** : 2×2 등 영역에서 대표값(Max 등)을 추출해 차원 축소

### 1.3 출력 크기 공식
```
H_out = ⌊(H_in - K + 2·P) / S⌋ + 1
```

예시 :
```
28×28 이미지, 필터 5×5, 패딩 0, 스트라이드 1
H_out = (28 - 5)/1 + 1 = 24   → 출력 24×24
```

### 1.4 컨볼루션 동작 예
3×3 이미지 패치 `[0,3,1,2,6,…]` 과 커널 `[1,2,3,4,5,6,7,8,9]` 의 element-wise 곱 합산:
```
0·1 + 3·2 + 1·3 + 2·4 + 6·5 + … = 31
```

### 1.5 Max Pooling
2×2 윈도우에서 최댓값 선택, stride 2 → H·W 절반으로 축소.
```
[[2, 6],      →  6  (Max)
 [3, 4]]
```
28×28 → 14×14 → 7×7 식으로 감소.

### 1.6 MNIST 실습 결과
- 단순 MLP 대비 CNN이 이미지 구조 정보를 잘 유지
- 약 **98% 정확도** 달성

## 2. RNN (Recurrent Neural Network)

### 2.1 핵심 개념
순차(sequence) 데이터(텍스트·음성·시계열)를 처리. 이전 시점의 **히든 상태(hidden state)** 를 현재로 전달하여 과거 정보 반영.

### 2.2 기본 수식
```
h_t = tanh(W_h · h_{t-1} + W_x · x_t + b)
y_t = W_y · h_t + b_y
```

### 2.3 기울기 소실 문제
시퀀스 길이 T가 길면 Chain Rule로 0~1 사이 미분 값이 반복 곱해져 **초기 시점 정보가 소실**. → 장기 의존성(long-term dependency) 학습 어려움.

## 3. LSTM (Long Short-Term Memory)

### 3.1 개요
게이트 메커니즘으로 중요한 정보를 장기간 유지.

### 3.2 게이트 수식
```
Forget Gate    : f_t = σ(W_f · [h_{t-1}, x_t] + b_f)
Input Gate     : i_t = σ(W_i · [h_{t-1}, x_t] + b_i)
Candidate Cell : c̃_t = tanh(W_c · [h_{t-1}, x_t] + b_c)
Cell State     : c_t = f_t ⊙ c_{t-1} + i_t ⊙ c̃_t
Output Gate    : o_t = σ(W_o · [h_{t-1}, x_t] + b_o)
Hidden State   : h_t = o_t ⊙ tanh(c_t)
```

### 3.3 게이트 의미
- **Forget** : 과거 셀 상태에서 어떤 정보를 버릴지
- **Input** : 현재 입력에서 어떤 새로운 정보를 저장할지
- **Output** : 셀 상태로부터 무엇을 다음 단계로 내보낼지

### 3.4 GRU (Gated Recurrent Unit)
조경현 교수가 제안한 LSTM 간소판. **리셋 / 업데이트** 2개 게이트만 사용 → 파라미터 감소·유사 성능.

## 4. N-gram 언어 모델

### 4.1 체인 룰 기반 확률 분해
```
P(w₁, w₂, …, wₙ) = P(w₁) · P(w₂|w₁) · P(w₃|w₁,w₂) · … · P(wₙ | w₁…wₙ₋₁)
                ≈ ∏ P(wᵢ | wᵢ₋ₙ₊₁ … wᵢ₋₁)
```

### 4.2 N-gram 종류
- **유니그램** : 단일 단어 확률만
- **바이그램** : 직전 1단어만 조건
- **트라이그램** : 직전 2단어 조건
- 한계 : 장거리 의존성 포착 불가 → RNN/Transformer로 극복

### 4.3 평가 지표 : Perplexity
```
Perplexity = exp(평균 cross-entropy loss)
```
- 낮을수록 언어 모델이 확신 있게 예측
- 높을수록 불확실

## 5. 실습 포인트
- MNIST(손글씨) : 28×28×1
- MSCOCO : 80개 클래스 이미지 분류 데이터
- 데이터 증강(Augmentation) : 회전·색상 변조 → 일반화 능력 향상
- BatchNorm / Dropout : 과적합 완화

## 6. 핵심 정리

1. CNN 출력 크기 공식 : `(H - K + 2P)/S + 1`.
2. 필터 개수 ↑ → 다양한 특징 추출, 파라미터 ↑.
3. 패딩 1 / 커널 3 / 스트라이드 1 → 입력 크기 유지.
4. Max Pool 2×2 stride 2 → 공간 크기 ½, 정보 75% 제거.
5. CNN = 로컬 연결 + **가중치 공유** → 파라미터 효율.
6. RNN은 가변 길이 시퀀스 처리 가능 (W 반복 사용).
7. RNN의 핵심 문제 = Vanishing Gradient.
8. LSTM 3게이트(Forget/Input/Output) + 셀 상태.
9. LSTM 셀 상태가 "곱하기 1"로 그래디언트 흐름 유지.
10. GRU는 LSTM 단순화, 2게이트.
11. N-gram 은 제한된 n-1 단어만 문맥 반영.
12. Perplexity로 언어모델 성능 평가.
13. MNIST + CNN ≈ 98% 달성.
14. 데이터 증강으로 일반화 향상.
15. BatchNorm·Dropout이 대표 정규화 기법.

### 강의 중 주요 직관 비유
- **"코가 여기 있든 저기 있든 코지요"** : CNN의 **Shift Invariance (이동 불변성)** 직관.
- **시험 공부 비유** : "문제은행만 풀지 말고 교양을 넓게 쌓아야 한다" — Pre-training + Fine-tuning 의 의미.
- **LSTM 셀 상태 = SSD** : 장기 메모리로 정보가 오래 유지되는 구조.
