# 260408 오후 강의 요약 — 딥러닝 혁신과 생성형 AI 개요

## 1. 딥러닝 = 자동 프로그래밍

**딥러닝** : 사람이 프로그램을 짜는 대신, 데이터를 주고 알고리즘이 스스로 파라미터를 학습하여 "프로그램"을 만드는 패러다임.
```
전통 프로그래밍 : (입력, 규칙) → 출력
머신러닝        : (입력, 출력) → 규칙(모델)
```

## 2. 딥러닝의 세 가지 혁신점

### 2.1 Layer-wise Sequential Pre-training
- **RBM(Restricted Boltzmann Machine)** 을 이용한 **층별 무감독 학습**
- 각 층을 학습하고 **freezing(고정)** 후 다음 층 추가
- Hinton(2006) 의 DBN이 대표

### 2.2 CNN (가중치 공유 + 로컬 연결)
- 동일한 필터를 이미지 전역에서 재활용 → **파라미터 수 수천~수만 배 절감**
- 200×200 RGB 이미지에서 64개 출력층 기준:
  - FC : 약 **768만 개** 파라미터
  - Conv 3×3 : 약 **4,800개** 파라미터 (≈1,600배 감소)

### 2.3 ReLU 활성화 함수
- `ReLU(x) = max(0, x)`
- Sigmoid의 미분값 ∈ (0, 0.25] → chain rule에서 계속 곱하면 0 수렴 (Vanishing Gradient)
- ReLU의 미분값 = 1 (x>0) → 깊은 신경망 학습 가능
- 2012 AlexNet 논문에서 ReLU의 빠른 수렴을 실험적으로 입증

## 3. Vanishing Gradient Problem 재조명

```
∂L/∂w_early = (∂L/∂h_last) · (∂h_last/∂h_prev) · … · (∂h₂/∂h₁) · (∂h₁/∂w_early)
```
- 각 미분값이 0~1 사이면 곱이 0으로 수렴
- 초기 층의 그래디언트가 거의 0 → 학습 불가
- 해결 : ReLU / Batch Normalization / Residual Connection / 적절한 가중치 초기화

## 4. 딥러닝의 세 가지 기본 구조

| 구조 | 학습 방식 | 모델 형태 | 대표 특징 |
|------|-----------|-----------|-----------|
| **CNN** | Supervised | Discriminative | 패턴 분류, 완전연결/부분연결 혼용 |
| **DBN (Deep Belief Network)** | Unsupervised + Fine-tuning | Generative | RBM 층층이 쌓기, Auto-encoder 계열 |
| **HyperNet** | Unsupervised (Lifelong) | Generative | 부분 연결, breadth 지향, 평생 학습 |

## 5. ImageNet과 AlexNet

### 5.1 ImageNet 챌린지
- 2010년 시작, 120만 장 이미지, 1,000개 클래스
- **2012 AlexNet** : 에러율 26.2% → **15.3%** (딥러닝 모멘트)
- 2015년 : **95% 정확도**로 인간 수준 도달
- 2013 DNN+HMM : 음성인식 성능 급상승

## 6. 생성형 AI 개요

### 6.1 Generative vs Discriminative
- **Discriminative** : `p(y|x)` — 결정 경계 학습
- **Generative** : `p(x)` 또는 `p(x|y)` — 분포 전체 학습, 샘플 생성 가능

### 6.2 Autoencoder
```
X → Encoder → 코드(z, 저차원) → Decoder → X' (X ≈ X')
```
- 입력 = 출력 목표 (Self-supervised)
- 압축-팽창 구조 → 자동 특징 추출
- 예: 100만 장 이미지 → 30차원 코드로 압축

### 6.3 GAN (Generative Adversarial Network)
```
Generator (z → 가짜 이미지)       Discriminator (진짜 vs 가짜 분류)
      ↓                                     ↑
      ─────── 경쟁(minimax game) ────────────
```
- **목적함수** : `min_G max_D  E[log D(x)] + E[log(1 - D(G(z)))]`
- Generator : Discriminator 를 속이는 쪽으로 학습
- Discriminator : 진짜/가짜 구분 성능 최대화
- 이상적 수렴 시 D ≈ 0.5

### 6.4 GAN Latent 공간의 벡터 연산
```
안경 쓴 남자 - 안경 없는 남자 + 여자 = 안경 쓴 여자
```
잠재 공간이 의미적(semantic) 구조를 학습함을 보여주는 대표 예.

## 7. 기계 번역 (Machine Translation)

### 7.1 Encoder-Decoder 구조
```
입력 문장 → Encoder → Context Vector → Decoder → 출력 문장
```

### 7.2 Attention의 필요성
- 긴 문장 하나를 단일 Context 벡터로 압축 → **Information Bottleneck**
- Attention : 각 출력 단어 생성 시 입력의 **어느 부분에 집중할지** 동적 결정 → 뒤 강의(Transformer)의 전신

## 8. 핵심 정리

1. 딥러닝 = 자동 프로그래밍, 패러다임 전환.
2. Vanishing Gradient는 chain rule 곱 누적에서 발생.
3. ReLU 미분 = 1 → vanishing 완화.
4. Pre-training(무감독) + Fine-tuning(감독) 이 레이블 부족 상황에 강력.
5. CNN의 가중치 공유로 파라미터 수 1000배↑ 감소.
6. 3기본 구조 : CNN / DBN / HyperNet.
7. AlexNet(2012) = 딥러닝 모멘트의 출발.
8. 2015년 ImageNet 95% → 인간 수준 도달.
9. Generative는 분포 전체, Discriminative는 경계만.
10. Autoencoder = 입출력 동일 목표의 self-supervised 학습.
11. GAN = Generator vs Discriminator minimax game.
12. GAN의 잠재 공간은 의미적 벡터 연산이 가능.
13. Encoder-Decoder는 seq2seq 문제의 일반 구조.
14. Attention은 Information Bottleneck을 해결하는 메커니즘.
15. 현대 LLM / Diffusion 등 모두 이 기초 위에 세워진다.

### 강의 중 주요 직관 비유
- **고양이 실험 (Hubel & Wiesel, 1960s)** : 특정 각도의 선에만 반응하는 뉴런 발견 → CNN 개념적 뿌리. 이후 노벨상 수상.
- **교양 교육 비유** : 초·중·고 다양한 과목 → 대학 전공 = **Pre-training + Fine-tuning** 의 인지적 은유.
- **"지피티는 세상의 모든 문서를 공부한 풍부한 상식의 전문가"** : 대규모 사전학습의 효과 설명.
