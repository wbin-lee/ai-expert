# 260416 오전 강의 요약 — Transformer 구조 상세 (Encoder 중심)

## 1. Transformer 전체 조감

### 1.1 구조 요약
```
       Input Tokens + Positional Encoding
               │
       ┌──── Encoder × 6 ────┐
       │  Multi-Head Self-Attn │
       │         Add & Norm    │
       │   Feed-Forward NN     │
       │         Add & Norm    │
       └───────────────────────┘
               │
       ┌──── Decoder × 6 ────┐
       │ Masked Self-Attention │
       │         Add & Norm    │
       │    Cross-Attention    │
       │         Add & Norm    │
       │   Feed-Forward NN     │
       │         Add & Norm    │
       └───────────────────────┘
               │
          Linear + Softmax
               │
         Output Probabilities
```

### 1.2 RNN 대비 이점
- **순차 의존성 제거** → 완전 병렬화 가능
- **Attention 으로 장거리 의존성 직접 연결** (Max path O(1))

## 2. Self-Attention 역할

### 2.1 문맥적 의미 학습
예: "I am so happy"
- `happy` 의 단어 임베딩(사전적 의미) 에 **문장 내 문맥** 을 더해 풍부한 표현 생성
- Self-Attention = **단어 임베딩 + 위치적·관계적 의미**

### 2.2 수식
```
Attention(Q, K, V) = softmax( Q · Kᵀ / √d_k ) · V
```

## 3. Multi-Head Attention

### 3.1 공식
```
head_i    = Attention( Q · W_iQ, K · W_iK, V · W_iV )
MultiHead = Concat(head₁, …, head_h) · W^O
```

### 3.2 설정 (원 논문 기준)
- `d_model = 512`
- `h = 8` (head 수)
- 각 head의 `d_k = d_v = 64`

### 3.3 효과
- 여러 head가 의미, 문법, 구조 등 **다양한 관계**를 병렬로 포착

## 4. Positional Encoding

### 4.1 필요성
Self-Attention은 permutation invariant → 순서 정보 0. 외부에서 위치 시그널 주입 필요.

### 4.2 Sinusoidal Encoding
```
PE(pos, 2i)   = sin( pos / 10000^(2i/d_model) )
PE(pos, 2i+1) = cos( pos / 10000^(2i/d_model) )
```
- 차원마다 다른 주파수
- **상대 거리**가 선형 변환으로 표현 가능

### 4.3 결합
```
최종 입력 = Token Embedding + Positional Encoding     (elementwise +)
```

## 5. Feed-Forward Network (FFN)

### 5.1 구조
```
Linear(d_model → d_ff) → ReLU → Linear(d_ff → d_model)
```
- `d_model = 512, d_ff = 2048` 원 설정
- **위치별 독립 적용** (Position-wise FFN)

### 5.2 역할
- Attention 결과에 **비선형 변환** 추가
- 토큰별로 같은 파라미터 공유

## 6. Add & Norm (Residual + LayerNorm)

### 6.1 Residual Connection
```
output = Sublayer(x) + x
```
깊은 네트워크에서 **그래디언트 흐름 보존**.

### 6.2 Layer Normalization
```
x̂ = (x − μ) / √(σ² + ε)
y  = γ · x̂ + β
```
- **배치 단위가 아닌 토큰 단위** 정규화
- 언어 모델에서 표준(BatchNorm 대비 안정)

## 7. Scaled Dot-Product Attention의 스케일링

### 7.1 √d_k 로 나누는 이유
d_k 가 클수록 `Q·Kᵀ` 분산이 커져 softmax 값이 saturate → 기울기 소실. 스케일링으로 softmax 분포 안정화.

### 7.2 결론
```
softmax( Q·Kᵀ / √d_k ) · V
```

## 8. 주요 설정값 (원 논문)

| 파라미터 | 값 |
|----------|-----|
| d_model | 512 |
| h (head 수) | 8 |
| N (layer 수) | 6 인코더 + 6 디코더 |
| d_ff | 2048 |
| d_k, d_v | 64 |
| dropout | 0.1 |

## 9. 핵심 정리

1. Transformer = Seq2Seq + Attention, 인코더·디코더 각 6층.
2. Self-Attention 으로 문맥적 의미를 학습.
3. Multi-Head로 다관점 관계 포착.
4. Scaled Dot-Product 공식 암기.
5. √d_k 로 softmax saturate 방지.
6. PE 없으면 위치 정보 0 ("duck bite man" = "man bite duck").
7. sin/cos 주기로 외삽 및 상대 거리 표현 가능.
8. 입력 = Token Emb + PE (element-wise 합).
9. FFN 은 위치별 독립 MLP.
10. Residual + LayerNorm 으로 깊이 안정화.
11. LayerNorm 은 토큰 단위 (언어 모델 표준).
12. 완전 병렬화 가능 → GPU 최적화에 매우 적합.
13. 원 논문 설정 : d_model=512, h=8, N=6, d_ff=2048.
14. Softmax 전의 값이 Logit.
15. Transformer가 Foundation Model (BERT·GPT·ViT) 의 표준.

### 강의 중 주요 직관 비유
- **아모레 화장품 아줌마 비유** : 1명이 8집 순차 방문(RNN) vs 8명이 동시 방문(Transformer).
- **위치 인코딩 냄새** : 기계가 수억 번 보면 "01001 냄새" 또는 "0.3425 냄새"로 위치를 인지한다는 은유.
