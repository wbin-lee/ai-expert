# 260416 오후 강의 요약 — Transformer Decoder · Masked Attention · Temperature

## 1. Decoder의 특이점

### 1.1 미래 토큰 문제
생성 시점 t에서는 t 이후의 토큰이 **아직 없음**. Encoder 처럼 양방향을 다 볼 수 없음 → **Masked Self-Attention** 필요.

### 1.2 세 가지 서브레이어
1. **Masked Multi-Head Self-Attention** — 현재·과거만 봄
2. **Cross-Attention** — Encoder 출력을 Q로 참조
3. **Feed-Forward Neural Net**

각 서브레이어 뒤에는 **Residual + LayerNorm**.

## 2. Masked Self-Attention

### 2.1 마스크 적용
Attention score 중 미래 위치를 `-∞` 로 설정 → softmax 후 0:
```
scores[i, j] = -∞   if j > i
softmax(scores) → 미래 위치의 확률 0
```

### 2.2 시각화
| 위치 | 1 | 2 | 3 | 4 |
|------|---|---|---|---|
| 1 | ✓ | × | × | × |
| 2 | ✓ | ✓ | × | × |
| 3 | ✓ | ✓ | ✓ | × |
| 4 | ✓ | ✓ | ✓ | ✓ |
- ✓ : 참고 가능 (과거/현재)
- × : 금지 (미래) — Forbidden City 비유

## 3. Cross-Attention

### 3.1 공식
```
Q from Decoder hidden state
K, V from Encoder output
Attention(Q, K, V) = softmax(Q·Kᵀ / √d_k) · V
```
Decoder가 각 생성 스텝에서 **Encoder의 모든 위치에 대해** 관련도를 평가해 정보를 끌어온다.

## 4. Language Modeling Head

### 4.1 출력 변환
```
decoder output (d_model) → Linear(d_model → |V|) → softmax → probability
```
- |V| : vocab 크기 (예 : 30,000)
- 각 위치에서 다음 토큰의 확률 분포 생성

### 4.2 Loss
```
L = - Σ_t  log P(y_t | y_{<t}, x)   # Cross-Entropy
```

## 5. Temperature Scaling

### 5.1 공식
```
P_i = exp(z_i / T) / Σ_j exp(z_j / T)
```

### 5.2 T 의 효과
| T | 분포 | 생성 성향 |
|----|------|-----------|
| 1 | 원본 | 표준 |
| > 1 | 평탄 | 다양성 ↑, Entropy ↑ |
| < 1 | 예각 | 결정적 |
| → 0 | One-Hot | Greedy |

### 5.3 예시
logits = [2.0, 1.0, 0.5]
- T=1.0 : [0.66, 0.24, 0.10]
- T=0.5 : [0.88, 0.10, 0.02] (집중)
- T=2.0 : [0.45, 0.33, 0.22] (확산)

## 6. Auto-Regressive Decoding과 Teacher Forcing

### 6.1 추론 (순차)
```
y_0 → y_1 → y_2 → … (각 단계 생성된 토큰을 다음 입력에 추가)
```

### 6.2 학습 (Teacher Forcing)
- 다음 입력으로 **정답 토큰**을 강제 투입
- 모든 위치를 **병렬 처리** 가능 → 학습 빠름
- Chain Rule : `P(y | x) = ∏_t P(y_t | y_{<t}, x)`

## 7. KV Cache (추론 최적화)

생성 시 이전 스텝의 K, V를 **캐싱** 해 반복 계산을 제거. 긴 문장 생성에서 큰 속도 이득.

## 8. 손실·역전파 흐름

1. Cross-Entropy Loss 계산 (정답 vs 예측 확률)
2. Decoder 6개 레이어 → Encoder 6개 레이어로 그래디언트 누적
3. Residual/Skip 덕분에 기울기 소실 완화
4. 배치 처리 시 다양한 길이의 문장은 패딩 후 처리

## 9. 핵심 정리

1. Decoder 1층 = Masked Self-Attn + Cross-Attn + FFN.
2. Masked Attention으로 미래 토큰 차단 (Causality 보장).
3. Mask 구현 = 미래 위치 score에 `-∞`.
4. Cross-Attention : Q=디코더, K/V=인코더.
5. LM Head = Linear(d→|V|) + softmax.
6. 학습 Loss = Cross-Entropy.
7. Teacher Forcing → 학습 병렬, 추론 순차.
8. Temperature로 다양성·결정성 조절.
9. T→0 = Greedy, T↑ = 창의적·위험.
10. Softmax 전 값 = Logit.
11. KV Cache로 추론 가속.
12. Skip Connection이 깊은 디코더 학습을 가능케.
13. 패딩 + Attention Mask 조합으로 배치 처리.
14. Decoder 정보는 Cross-Attention 을 통해 Encoder 표상을 참조.
15. 최종 생성은 `argmax` 또는 샘플링(Top-K/Top-P/Temperature).

### 강의 중 주요 직관·일화
- **폰 노이만 → 섀넌** : "엔트로피라고 하라, 아무도 모르니 우위에 선다" — 물리학 개념(엔트로피·온도)의 AI 차용.
- **Forbidden City (자금성)** : 미래 토큰 진입 금지를 시각적으로 표현.
- **온도·엔트로피 직관** : 높은 T → 더 무질서하고 다양 / 낮은 T → 질서 있고 결정적.
